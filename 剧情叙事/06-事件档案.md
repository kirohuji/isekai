# 事件档案

## 说明

本文档是**所有剧情事件的目录与详情档案**。每个事件都是一个独立的叙事单元——有自己的触发条件、核心冲突、关键选择、代价与收益、以及与其他事件的连接点。

> 不要把这个当成"关卡设计文档"。把它当成**叙事素材库**——写作时翻一翻，看到哪个事件适合当前节奏，就把它编排进去。

---

## 事件模板

每个事件按以下结构记录：

```
## [事件编号] [事件名称]

| 维度 | 内容 |
|---|---|
| **类型** | 生存/战斗/社会/经济/探索/感情/混合 |
| **阶段** | 已发生/进行中/待触发/可选支线 |
| **触发条件** | 需要满足什么前置条件才能触发 |
| **地点** | 发生在哪个区域 |
| **核心冲突** | 这个事件的核心矛盾是什么（一句话） |
| **关键角色** | 涉及的主要NPC |
| **男主的核心能力** | 这个事件考验男主的什么能力 |
| **关键选择** | 男主要做的重要决策 |
| **最低代价** | 即使最优解也要支付的代价 |
| **最大收益** | 成功的回报 |
| **失败后果** | 搞砸了会怎样 |
| **与其他事件的连接** | 完成后解锁/影响哪个事件 |
| **叙事节奏** | 紧张拍/过渡拍/人物拍 |
| **建议章节数** | 预计占用多少节 |
| **暗黑指数** | ★☆☆☆☆ ~ ★★★★★ |
| **写作注意** | 写作时需要注意的特殊要点 |
```

### E-001 灰绳市场买奴

| 维度 | 内容 |
|---|---|
| **类型** | 社会/感情 |
| **阶段** | ✅ 已发生（第一章第一节） |
| **触发条件** | 男主有足够银币 + 灰绳市场开放 + 市场上有合适的目标 |
| **地点** | 王都城外·灰绳市场 |
| **核心冲突** | 用有限的银币，在众多奴隶中选择一个"最有用"的——但选择的标准是什么？价格？技能？还是那张让他想起过去的脸？ |
| **关键角色** | 沈清岚（重伤、女扮男装）、奴隶贩子沃尔克 |
| **男主的核心能力** | 价值判断、成本计算、在信息不对称中做决策 |
| **关键选择** | 1) 买不买 2) 买谁 3) 花多少钱。男主选择了8银币买下重伤的沈清岚——不是因为认出她，是因为"便宜+像过去暗恋的人" |
| **最低代价** | 8银币（约等于一个月的旅馆分期款） |
| **最大收益** | 沈清岚——掌握50人名单、神殿文字、预言记录、以及后续成为男主最信任的伙伴 |
| **失败后果** | 钱花了人死了（她当时重伤高烧）——8银币打水漂 |
| **与其他事件的连接** | 直接触发 E-003（藏匿与照料）、间接连接 E-007（沈清岚告白） |
| **叙事节奏** | 人物拍→紧张拍（穿越贫民窟、进入市场） |
| **建议章节数** | 1节 |
| **暗黑指数** | ★★★★☆ |
| **写作注意** | 奴隶市场的环境描写——不要只写"脏"和"惨"。写出"制度化"的恐怖：讨价还价的人、旁边在买菜的主妇、笼子里的人看着路过的每个人。最可怕的是所有人都觉得这很正常 |

### E-002 旧鹿角旅馆接手

| 维度 | 内容 |
|---|---|
| **类型** | 经济 |
| **阶段** | ✅ 已发生（第一章第二节·回忆） |
| **触发条件** | 男主在王都扛麻袋攒钱→观察到旧鹿角旅馆→与前任酒鬼老板谈判 |
| **地点** | 王都平民区北部·旧鹿角旅馆 |
| **核心冲突** | 一个无能力的异世界人，如何在没有任何信用记录的情况下接手一桩濒临倒闭的生意 |
| **关键角色** | 前任老板（酒鬼）、玛莎（后来的厨娘） |
| **男主的核心能力** | 观察力（发现旅馆的真正价值被老板的个人问题掩盖）、谈判、分期付款的财务设计 |
| **关键选择** | 分期接手——不是一次性买断，而是用未来收入偿还。这个选择让男主绑定了旅馆，也绑定了债务 |
| **最低代价** | 首付 + 每月8银币分期 + 40银币总债务 |
| **最大收益** | 6间客房的旅馆 + 15亩农田 + 仓库 + 骡车。一个独立的据点和合法身份 |
| **失败后果** | 无法偿还分期→失去旅馆→回到扛麻袋的生活 |
| **与其他事件的连接** | 旅馆是 E-003、E-005、E-006 的场景基础 |
| **叙事节奏** | 过渡拍 |
| **建议章节数** | 1节（作为回忆穿插） |
| **暗黑指数** | ★★☆☆☆ |
| **写作注意** | 这一节的精彩之处在于男主的观察过程——他不是"幸运地"找到了旅馆，而是通过长时间的观察、比较、计算，做出了一个理性的投资决策。这定义了"男主的优势是脑子" |

| **触发条件** | E-001 完成后 |

| **最低代价** | 暴露风险 + 玛莎知道了（多一个人知道就多一分风险） |

| **暗黑指数** | ★★★☆☆ |
| **写作注意** | 男主的性侵犯冲动在这里出现——需要写出：他有这个念头→他用理智战胜→这不是"他变好了"，而是"他本来就是有阴暗面的人，但他选择了不实施"。道德不在于没有邪念，而在于面对邪念时的选择 |

### E-004 集市侦察

| 维度 | 内容 |
|---|---|
| **类型** | 探索/社会/感情 |
| **阶段** | ✅ 已发生（第一章第四节） |
| **触发条件** | 沈清岚恢复至能走四条街 + 男主需要确认她是否能独立行动 + 市场信息需要更新 |
| **地点** | 王都·北门集市 |
| **核心冲突** | 在暴乱前夕的紧张氛围中，带着一个"不能被人多看"的人出门——既是侦察任务，也是两人第一次在公开场合以"非主奴"方式相处 |
| **关键角色** | 沈清岚、卖衣摊主（告密者）、粮铺老板 |
| **男主的核心能力** | 路线规划（避开神殿检查岗）、信息收集（通过沈清岚的观察获取市场情报） |
| **关键选择** | 给她买外套。这个选择的情感重量远大于十铜币——这是男主第一次"为她花钱但不算账" |
| **最低代价** | 十铜币 + 暴露风险（摊主后来向神殿告密） |
| **最大收益** | 发现粮食和银叶草被预购的异常（暗示有人在囤积）+ 沈清岚开始展现她的情报分析能力 + 两人的关系从"主奴"向"伙伴"迈出第一步 |
| **失败后果** | 被神殿检查岗拦截→提前暴露→没有后面的搜查应对 |
| **与其他事件的连接** | 直接为 E-005（神殿搜查）提供信息基础（沈清岚知道契约的漏洞、搜查铃的原理）；也为后续的粮食囤积线索（罗德里克线）埋下伏笔 |
| **叙事节奏** | 人物拍（买衣服时的互动）→紧张拍（暴乱爆发、逃回旅馆） |
| **建议章节数** | 1节 |
| **暗黑指数** | ★★☆☆☆ |
| **写作注意** | 摊主告密是这一节最精妙的设计——她不是"反派"，她只是一个在暴乱中为了自保和赚点额外的钱而出卖信息的普通人。恶不需要邪恶的动机 |

### E-005 神殿搜查

| 维度 | 内容 |
|---|---|
| **类型** | 社会/生存 |
| **阶段** | ✅ 已发生（第一章第五、六节） |
| **触发条件** | 沈清岚被神殿追踪 + 摊主告密提供了线索 + 暴乱导致神殿加强搜查 |
| **地点** | 旧鹿角旅馆 |
| **核心冲突** | 神殿带着异世界检测铃登门搜查——男主必须用制度和信息漏洞（而非武力）保护一个藏在墙里的人 |
| **关键角色** | 沈清岚、记录官（蓝袍）、两个灰袍守卫、玛莎、莱恩 |
| **男主的核心能力** | 制度漏洞利用（用登记过的召唤者身份干扰检测铃）、压力下的表演、信息不对称的操作 |
| **关键选择** | 1) 信不信任沈清岚的方案（用异界物品干扰铃） 2) 交出"异界物品"还是抵抗 3) 面对契约被拿走时的反应。男主选择了交出物品、接受"观察点"标签——以空间换时间 |
| **最低代价** | 手机+学生证+校徽被没收（男主与故乡的物质联系被切断）；旅馆被标记为"观察点"（长期监视）；契约被拿走（沈清岚的合法身份证明消失）；街对面多了两个监视者 |
| **最大收益** | 沈清岚没被发现。搜查队暂时离开。男主和沈清岚在压力下建立了"共谋"的默契 |
| **失败后果** | 夹墙被发现→沈清岚被抓→男主以"窝藏神殿逃犯"被处决或终身监禁→旅馆员工连坐 |
| **与其他事件的连接** | 搜查结果直接触发 E-006（共同决定）——因为"不能再把她塞回夹墙"和"契约没了"迫使两人必须做出下一步决策 |
| **叙事节奏** | 紧张拍（铃舌转动、剑鞘敲墙） |
| **建议章节数** | 2节（第五节=准备与博弈前半、第六节=搜查高潮与后遗症） |
| **暗黑指数** | ★★★☆☆ |
| **写作注意** | 记录官不是"凶神恶煞的反派"。他客气、礼貌、按流程办事——这种"制度化的恶"比个人的恶更可怕。他不是来"害"男主的，他是来做他的工作。铜铃和蜡印是"制度"的物质化身 |

### E-006 共同决定

| 维度 | 内容 |
|---|---|
| **类型** | 感情/社会 |
| **阶段** | ✅ 已发生（第一章第七节） |
| **触发条件** | E-005 完成（搜查结束、契约被拿走、旅馆被监视） |
| **地点** | 旧鹿角旅馆·厨房 |
| **核心冲突** | 男主一直以来的模式——"我保护你、我替你做决定"——被沈清岚正面挑战。她不要被保护，她要共同决定 |
| **关键角色** | 沈清岚 |
| **男主的核心能力** | 自我反思——面对自己"保护欲=控制欲"的事实，选择改变行为模式 |
| **关键选择** | 是继续"我来想办法"的老路，还是接受"我们一起定"的新规则。男主选择了后者 |
| **最低代价** | 放下控制权——这意味着他不再能单方面决定沈清岚的生死风险。这是一种**信任的代价** |
| **最大收益** | 两人关系从"保护者-被保护者"升级为"平等伙伴"。这个关系基础将支撑后续所有共同行动。同时沈清岚提出了利用病证+转运路线的方案——这是两人"共同决策"的第一个产物 |
| **失败后果** | 如果男主拒绝改变→沈清岚对他的信任将不可逆地破裂→后续所有关系发展失去基础 |
| **与其他事件的连接** | 两人的新关系模式将直接影响 E-008（四层撤离）、E-012（沈清岚告白）等所有后续事件 |
| **叙事节奏** | 人物拍——全章最安静但最重的一节 |
| **建议章节数** | 1节 |
| **暗黑指数** | ★★☆☆☆ |
| **写作注意** | 这一节的精髓在于"不说出来的比说出来的多"。沈清岚说"你现在做的事，跟他们很像"——这句话比任何告白都重。男主的回忆闪回（高中器材室）是第一人称意识流的典范运用——过去的记忆在当下的时刻被激活，让他理解了"站在门外"和"参与其中"之间的区别 |

### E-007 沈清岚告白

| 维度 | 内容 |
|---|---|
| **类型** | 感情 |
| **阶段** | ✅ 已发生（见时间线·第10阶段） |
| **触发条件** | 逃亡过程中积累的信任 + 男主写下放归文书 + 她确认男主不会利用契约控制她 |
| **地点** | 沼泽途中 |
| **核心冲突** | 她承认过去不喜欢男主（甚至曾喜欢周启明），但说明现在喜欢的是男主。男主不立即接受也不拒绝 |
| **关键角色** | 沈清岚、男主 |
| **男主的核心能力** | 面对感情的诚实——不因为"她告白了"就匆忙接受 |
| **关键选择** | 不接受、不拒绝、写下放归文书。这个"中间态"比直接接受或拒绝更需要勇气 |
| **最低代价** | 感情线的模糊状态将持续带来张力 |
| **最大收益** | 放归文书——沈清岚的法律自由尝试 |
| **失败后果** | 如果男主拒绝→不会导致关系破裂（沈清岚说过"不逼迫"），但会改变两人之间的空气 |
| **与其他事件的连接** | 直接连接到 E-013（后续感情发展） |
| **叙事节奏** | 人物拍 |
| **建议章节数** | 1节 |
| **暗黑指数** | ★☆☆☆☆ |
| **写作注意** | 不要写成"浪漫告白场景"。写成一个"她需要说出来"的时刻——不是为了得到回应，是因为在随时可能死掉的逃亡途中，有一些话不说可能就永远没机会了 |

### E-009 周启明来信与赴约拒绝

| 维度 | 内容 |
|---|---|
| **类型** | 社会/心理 |
| **阶段** | ✅ 已发生（见时间线·第7阶段） |
| **触发条件** | 周启明推断出沈清岚在旅馆 + 他选择先不告密而是发信 |
| **地点** | 旧鹿角旅馆→旧钟楼（约定地点，男主未去） |
| **核心冲突** | 周启明知道了。他的信说"来旧钟楼"。男主必须判断：这是陷阱？是真的想谈？还是两者都是？ |
| **关键角色** | 周启明（未出场，仅通过信件和传闻存在） |
| **男主的核心能力** | 风险判断——识别"在对方设定的时间和地点见面"的战略劣势 |
| **关键选择** | 不去。这是男主在故事中最重要的决策之一：不被对方设定的规则牵着走 |

| **关键选择** | 1) 接受神殿册封（获得合法性但被绑上神殿体系） 2) 寻求国王承认（绕过神殿但可能得罪神殿） 3) 依靠"事实控制"（不追求合法头衔但更脆弱） 4) 接受铁棘公国的暗中支持（背叛圣光王国但获得外部保护） |
| **最低代价** | 无论选择哪条路，都需要付出政治代价——站队意味着树敌 |
| **最大收益** | 被承认的领地——可以合法征兵、征税、与其他势力进行外交 |
| **失败后果** | 被视为"非法武装据点"→被镇压 |
| **与其他事件的连接** | 领地的合法性决定了 E-020（红月前夕的最终选择）中男主有多少筹码 |
| **叙事节奏** | 过渡拍（政治博弈）→紧张拍（如果选择与某一方对抗） |
| **建议章节数** | 持续多章，不是单一事件而是渐进过程 |
| **暗黑指数** | ★★★☆☆ |
| **写作注意** | 领地合法性不是"打一场胜仗就获得了"。是漫长的政治博弈——谈判、妥协、站队、背刺。男主在这个过程中可能做出他自己都厌恶的选择 |

### E-019 救援同学行动

| 维度 | 内容 |
|---|---|
| **类型** | 社会/策略/感情 |
| **阶段** | 🟢 远期 |
| **触发条件** | 灰丘有足够资源（粮食、人力、药品）+ 沈清岚名单中有"可救援+值得救援"的目标 + 男主阵营有执行救援的武力 |
| **地点** | 取决于目标同学的位置——可能是王都、矿区、神殿庄园、某个边境小镇 |
| **核心冲突** | 救不救？救谁？怎么救？每个被救的同学都带来新的人力——也带来新的嘴要喂、新的风险、新的人际关系。被救的人可能不感激——有人会怨恨"你怎么现在才来" |
| **关键角色** | 目标同学（性格、能力、立场待定）+ 沈清岚（名单信息来源）+ 格兰（武力支持）+ 救援对象所在地的地方势力 |
| **男主的核心能力** | 成本收益评估——"可以救援"≠"必须救援"。道德取舍——放弃某些人 |
| **关键选择** | 1) 先救谁（能力强者？关系近者？近者？） 2) 冒多大风险 3) 救回来后怎么安置 |
| **最低代价** | 资源消耗（每次救援=粮食+药品+人力+风险） + 暴露灰丘位置 + 被救者的"附带问题"（怨恨、疾病、被追踪） |
| **最大收益** | 新的人力（特别是治愈系能力者如苏瑶）+ 情报（被救者知道神殿/矿区的内部信息） + 感情线推进 |
| **失败后果** | 救援失败→男主阵营暴露→被神殿锁定→灰丘被连根拔起 |
| **与其他事件的连接** | 救援苏瑶可能触发治愈系感情线。救援其他同学可能揭示神殿内部情报 |
| **叙事节奏** | 策略拍（选择+计划）→紧张拍（执行）→人物拍（被救者的反应、融入过程） |
| **建议章节数** | 每次救援2-3节。可多次执行（分阶段救援不同人） |
| **暗黑指数** | ★★★★☆ |
| **写作注意** | 第一次救援应该以"几乎失败"告终——男主低估了难度。被救者的"不感激"是真实的人物反应——写出来比写"感谢救命之恩"更有力量。男主必须在"救更多人"和"保护已有的人"之间做出痛苦的选择 |

## 四、可选支线事件（可插入任何合适的位置）

### E-020 逃亡矮人铁匠

| 维度 | 内容 |
|---|---|
| **类型** | 探索/社会 |
| **阶段** | 🟢 可选支线 |
| **触发条件** | 通过格兰或鼠族商队得知某个逃亡矮人在边境隐姓埋名 |
| **地点** | 暮河镇或苇水村附近的某个隐蔽工坊 |
| **核心冲突** | 一个逃亡矮人铁匠——他的技能对灰丘的防御建设至关重要，但他的存在本身就是"窝藏逃奴"的罪名。帮他=风险，不帮=失去最好的武器来源 |
| **关键角色** | 逃亡矮人、格兰、鼠族商队中间人 |
| **男主的核心能力** | 谈判——矮人不信任人类。如何用"交易"而非"施舍"建立关系 |
| **关键选择** | 1) 是否庇护矮人 2) 用什么交换他的锻造服务 |
| **最低代价** | 庇护逃奴的法律风险 + 矮人需要的物资 |
| **最大收益** | 矮人锻造的武器和工具 + 地下通道知识 |
| **失败后果** | 矮人被追捕队发现→男主被牵连 |
| **叙事节奏** | 人物拍（矮人的故事）→过渡拍（建立合作关系） |
| **建议章节数** | 1-2节 |
| **暗黑指数** | ★★★☆☆ |
| **写作注意** | 矮人不是"快乐的铁匠"。他是被奴役了半辈子后逃出来的——手指有旧伤、沉默寡言、对人类有深度不信任。不是因为男主"善良"打动了他——是因为男主提供了他无法拒绝的条件（保护他的家族？帮他传递消息给还被困在矿区的族人？） |

### E-021 魅魔遭遇

| 维度 | 内容 |
|---|---|
| **类型** | 探索/社会/暗黑 |
| **阶段** | 🟢 可选支线 |
| **触发条件** | 男主阵营在边境遇到一个"看起来需要帮助的独身女性"——她是伪装形态的魅魔 |
| **地点** | 边境地带（暮河镇外围、沼泽边缘或某个废弃村落） |
| **核心冲突** | 这个女的是魔族的间谍。但她也在"选"——她可以把这个人类阵营出卖给魔将，也可以利用他们来逃脱魔族的控制。她不是"善良的魅魔"，她是"在计算哪个选择对自己更有利的魅魔" |
| **关键角色** | 魅魔（伪装形态——外貌、姓名、背景都是假的）、格兰（亚人的嗅觉可能察觉异常——但不一定） |
| **男主的核心能力** | 识别伪装——通过细节（她的体温偏低？她的影子偶尔不对劲？她对某些神殿物品的反应？）判断她的真实身份。以及判断：一个"不完全是敌人"的魔族是否有利用价值 |
| **关键选择** | 1) 揭穿她 2) 利用她（套取魔族情报） 3) 驱逐她 4) 杀了她。但她也可能有自己的算盘——在被揭穿后提出一个交易 |
| **最低代价** | 暴露灰丘位置给魔族（如果她选择出卖） |
| **最大收益** | 魔族内部情报（魔将的数量、魔王的真相、魅魔的生存困境） |
| **失败后果** | 被魅魔吸取精力（如果她选择攻击）→男主或格兰虚弱→另一个危机中无法自保 |
| **叙事节奏** | 人物拍（她的"求助"表演）→紧张拍（真相揭露的时刻） |
| **建议章节数** | 2-3节 |
| **暗黑指数** | ★★★★★ |
| **写作注意** | 魅魔不能写成"性感恶魔爱上男主"的轻小说套路。她的吸引力不在于色情——在于她是一个和你一样"被困在系统中的聪明人"。如果男主帮她，不是因为"爱上她"，是因为"一个魔族叛逃者是最有价值的情报源"。魅魔的精力吸取需要——这是一个无法回避的生存问题。她会吸谁？她怎么吸？这些问题的黑暗面必须被直面 |

### E-022 红月重合夜（常驻事件）

| 维度 | 内容 |
|---|---|
| **类型** | 气氛/过渡 |
| **阶段** | 🔄 每78天触发一次 |
| **触发条件** | 每78天——无论剧情进行到哪里 |
| **地点** | 任何地点——所有角色都能看到天空 |
| **核心冲突** | 这不是一个"事件"——是**常驻的倒计时提醒**。每78天一次，天空变成红色。所有人停下手中的事，抬头看。这一次比上一次更红了吗？ |
| **关键角色** | 所有角色。每个人的反应不同：米娅害怕、沈清岚计算下一次重合的日期、格兰无动于衷（他说"亚人见过更红的"——但他在说谎）、男主在红色天空下做某个决定 |
| **男主的核心能力** | 无。红月不考验任何能力——它只是在提醒：时间在走 |
| **关键选择** | 无具体选择。但每次红月重合夜——男主的心理状态、他在那一刻做出的决定——被红色天空压上一层重量 |
| **最低代价** | 心理压力 |
| **最大收益** | 叙事节奏的天然节点——适合作为卷的结尾或重大事件前的预兆 |
| **失败后果** | 无 |
| **与其他事件的连接** | 每次重合都可以与当时正在进行的其他事件叠加——在大事件发生的同时，天空正变成红色 |
| **叙事节奏** | 气氛拍 |
| **建议章节数** | 0.5节——每次重合不需要一整节，但应该在某个场景中穿插"今晚是红月重合夜" |
| **暗黑指数** | ★★★★☆ |
| **写作注意** | 不要每次都写长篇的红月描写。第一次要详细（建立意象）。之后的每一次只需要一句话——"她抬头看了一眼窗外。天空又红了一点。"就够了。读者已经知道这意味着什么 |

## 六、使用方式

1. **写作前**：查看"待触发事件"列表，选择适合当前节奏的事件
2. **写作中**：参考事件的"关键选择""最低代价""写作注意"——确保不偏离设计意图
3. **写作后**：如果事件的实际走向与设计不同——**更新此文档**，以实际写作为准
4. **规划时**：使用事件关系图谱确认事件间的依赖关系——不要在B事件的前置条件满足之前就写B事件
