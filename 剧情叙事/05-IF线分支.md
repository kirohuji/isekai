# IF线分支——不同的起点，不同的人生

## 说明

本文档收录男主在**召唤后最初的关键决策节点**上做出不同选择所引发的**分支故事线**。

正史线（当前故事）的路径是：**安置金→学语言→买旅馆→买沈清岚→逃亡→灰丘。**

每条IF线从"**决策分歧点**"开始分叉，描述由此展开的不同故事走向、主要角色关系、以及感情线。

> IF线不取代正史线。它们是世界观和角色的"压力测试"——如果男主选择了另一条路，这个世界的哪些部分会暴露出来？

---

## 决策分歧点：120银币怎么用

召唤后获得120银币安置金。这是男主在异世界掌握的第一笔——也是最后一笔——自由支配的资本。**怎么花这120银币，决定了他在这个世界的全部轨迹。**

---

## 正史线（当前故事）：经营独立路线

| 维度 | 内容 |
|---|---|
| **关键决策** | 20银币学语言 → 从事物流生意积累利润 → 分期接手旧鹿角旅馆 → 买下沈清岚（8银币） |
| **核心优势** | 独立身份、小规模商业网络、旅馆作为安全据点 |
| **核心劣势** | 无武力、无政治庇护、孤军奋战 |
| **感情线** | 沈清岚（第一女主）→ 蕾娜（潜在第二）→ 伊丝琳（潜在第三）→ 苏瑶（潜在第四） |
| **故事基调** | 逃亡+经营+末日生存 |

## IF-A线：神殿合作路线——「档案室里的共谋者」

### 分歧点

男主没有花20银币学语言，也没有做生意。他选择**回应神殿的"观察"**——主动接触神殿，表示愿意配合。

### 故事走向

| 阶段 | 内容 |
|---|---|
| **神殿下注** | 神殿对男主的评估是"暂不接触，持续观察"。男主主动上门——让神殿决定"那就接触，看他能做什么"。他被分配到第三圣殿做档案管理辅助——因为神殿认为一个无能力但有组织能力的人，最适合放在"信息处理"而不是"武力部门" |
| **发现秘密** | 在整理档案的过程中，男主**比沈清岚更早**发现召唤记录的异常——召唤并非第一次发生。但他没有逃跑——他在系统内部**收集证据** |
| **遇到艾琳** | 他的直属上司是低级女祭司艾琳（见 `女性角色与感情线.md`）。她表面温顺，实际也在暗中收集神殿的秘密——因为她在档案中见过太多"不该存在"的东西。两人一开始互相试探，然后慢慢发现：对方和自己在做同一件事 |
| **双重生活** | 白天：虔诚的档案管理员。晚上：秘密记录、复制文件、寻找可以信任的人。他们在系统内部建立了一个微型的抵抗网络——成员包括一个药剂师学徒（帮过玛莎的那位）、一个底层修士、一个负责打扫圣殿的聋哑老人 |
| **外部冲突** | 周启明等勇者可能发现男主的"异常积极性"。第四圣殿的异端审判机构可能嗅到"内部不纯"的气味 |
| **红月倒计时** | 当红月预言出现，男主和艾琳必须做出选择：带着收集到的证据逃离神殿，还是留在系统内部试图阻止即将到来的灾难——以两个平民的身份对抗整个神权体制 |

### 感情线

| 女主 | 说明 |
|---|---|
| **艾琳（神殿女祭司）**——主 | 同盟→共谋→感情。两人的关系建立在"信息的秘密交换"之上。每一次靠近都有风险——任何暴露都意味着两人都会被送上异端审判 |

### 本线独有的叙事张力

- **系统内的反抗**：不是逃亡者，而是渗透者。最大的敌人不是"会不会被抓"，而是"在系统内待久了，你会不会变成系统的一部分"
- **信息即武器**：没有战斗场面。所有的"战斗"都在档案室、走廊对话和深夜的密室里进行
- **艾琳的信仰**：她不相信神殿的官方叙事——但她信仰女神本身。当男主开始怀疑女神是否真实存在——这是两人的关系中最深的裂痕

## IF-B线：亚人之恋路线——「八枚银币，一个狐耳奴隶」

### 分歧点

男主拿到120银币后，学语言、做小生意——但**没有买旅馆**。而是将全部积蓄用来在灰绳市场买下了一个亚人女性奴隶——不是为了性，是因为在当时，这是他觉得"最可能帮助自己活下去的投资"。

> 他没有遇到沈清岚。他遇到的是卡莎。

### 故事走向

| 阶段 | 内容 |
|---|---|
| **购买** | 在灰绳市场，他看到了卡莎——一个狐系兽耳族年轻女性，脸上有奴隶印记。她在主动向买家展示自己的记账能力。男主用远高于8银币的价格（约50银币——一个健康且有技能的亚人奴隶的市场价）买下了她。这是他全部的流动资金 |
| **互相试探** | 卡莎判断：这个买家不像是买奴隶的人。他问她"你会记账"时的语气不像是在评估财产，像是在评估求职者。她决定观察——同时保持最高警惕 |
| **合作经营** | 男主和卡莎在王都做小规模的物资倒卖。卡莎的狐系社交天赋+男主的物流头脑——他们在暴乱前的王都赚到了钱。但她作为亚人奴隶的身份限制了他们能进入的所有地方 |
| **逃亡** | 红月预言出现。王都暴乱。男主决定带卡莎逃往东南。卡莎选择跟他走——不是因为她"喜欢"他，而是因为她判断他是自己活着离开王都的最佳机会 |
| **亚人网络** | 在边境，卡莎的亚人身份从"负担"变成了"资产"。她能用狐系社交天赋与亚人聚落沟通。男主通过她进入了亚人网络——不是作为"格兰的合作伙伴"，而是作为"卡莎的人类" |
| **信任还是感情？** | 核心难点：卡莎从一开始对男主的"信任"就是计算的一部分。她自己分不清自己对他的依赖是"策略"还是"真的"。而他也不知道——他给她自由的时候，她是真的想留下，还是觉得"离开后的风险大于留下" |

### 感情线

| 女主 | 说明 |
|---|---|
| **卡莎（狐系亚人奴隶）**——主 | 从"计算"到"真实"的感情进化。这是最黑暗的感情线——因为奴隶制扭曲了人与人的一切关系。两人需要在一片废墟中重建"信任"的定义 |

### 本线独有的叙事张力

- **亚人视角**：男主被人类的身份定义，被推入亚人网络生存。他从"人类的压迫者立场"转到"被迫在人类和亚人之间选择"——这不是道德选择，是生存选择
- **奴隶制的心理后遗症**：卡莎的创伤不是"被打"，而是"不知道自己的微笑是自愿的还是演出来的"。她的感情弧线是"重新学习什么是自愿的"
- **没有沈清岚**：正史线中沈清岚的知识（名单、神殿秘密）在本线中不存在。男主对神殿的真相完全无知——这意味着他可能在红月日之前对末日毫无准备

## IF-C线：佣兵路线——「把后背交给一个杀过人的人」

### 分歧点

男主拿到120银币后，用50银币购置基础装备（皮甲、短刀、药包），用剩余的钱在佣兵工会注册并加入一支小型佣兵团。从后勤做起——搬运、记账、清理战场。

### 故事走向

| 阶段 | 内容 |
|---|---|
| **底层佣兵** | 男主在佣兵团里是最底层——无神力者，没有战斗经验，连装备都是二手的。他在佣兵团里负责别人都不想做的事情：记账、物资管理、在战斗后清理死者的遗物 |
| **遇到维拉** | 维拉（见 `女性角色与感情线.md`）是佣兵团中最有经验的战士之一——也没有神力。她对男主的最初态度是："又一个来了就会死的"。但男主在第一次战斗中的表现——不是勇敢，而是冷静——让她多看了一秒 |
| **佣兵生涯** | 两人的关系在战场上建立。不是爱情——是"我守你的左边，你守我的右边"的默契。维拉开始教男主基础的战斗技巧——不是为了让他变强，是为了"别让我在一个蠢货旁边战斗" |
| **佣兵团的黑暗面** | 佣兵团接受的任务越来越黑暗——镇压矿区、追捕逃奴。男主面临选择：继续跟着佣兵团拿钱（生存），还是拒绝参与（但可能被佣兵团抛弃——在佣兵团里，不服从的人就是敌人） |
| **分裂与逃亡** | 维拉告诉他：她也曾面临同样的选择。她选择的是沉默——然后悔恨。当男主决定拒绝参与镇压任务时，维拉选择了跟男主一起离开佣兵团。两人被佣兵团追捕——同时还要在末日中求生 |

### 感情线

| 女主 | 说明 |
|---|---|
| **维拉（女佣兵）**——主 | 战友情→感情。这是最"硬"的感情线——没有告白，没有浪漫。所有的感情都在战斗动作里：她教他的持刀姿势、战斗中她站在他的左侧、受伤后她用比他更粗暴的方式给他包扎 |

### 本线独有的叙事张力

- **暴力对人的侵蚀**：男主在这条线中会杀人。不是象征性地"在危机中自卫"——是在佣兵生涯中杀人。这对男主的心理影响是正史线中不存在的东西
- **无神力者的反抗**：维拉和男主都没有神力。他们代表了这个世界中被"能力等级制度"抛弃的人如何自己争取生存权
- **佣兵团的道德灰色地带**：每个人的手都不干净。维拉自己就杀过不该杀的人。感情在这样的背景下——不是救赎，是两个有罪的人互相支撑

## IF-D线：商会路线——「用银币买位子，用情报买命」

### 分歧点

男主拿到120银币后，没有买任何实体资产。他用全部资金**购买信息和关系**——贿赂商人公会的小吏、在王都商业区租一个不起眼的摊位、建立物流和情报网络。

> 他是异世界50人中唯一一个在商人公会登记的人。

### 故事走向

| 阶段 | 内容 |
|---|---|
| **情报商起家** | 男主的"商品"不是货物——是信息。他知道哪些同学被分到了哪里（从沈清岚的名单中获得——但如果沈清岚不存在……则从商人公会的信息渠道中拼凑）。他知道王都的粮价什么时候会涨。他知道哪条路是安全的。这一切信息都可以卖钱 |
| **遇到莉迪娅** | 莉迪娅（见 `女性角色与感情线.md`）是商人公会核心成员的女儿——也是实际经营家族粮食贸易的人。她最初注意到男主是因为——这个异世界来的穷小子，怎么比本地商人还懂物流？ |
| **对等的商业关系** | 男主和莉迪娅的关系不是"贫穷但努力的男人+富有但被感动的女人"。是**两个精于算计的人在互相试探之后发现，对方是自己能找到的最好的合作伙伴**。感情是从生意伙伴关系中萌生的——不知从什么时候开始，她在谈判中会给他多留一个点的利润 |
| **体制内的博弈** | 男主在商会体系中一步步往上爬——不是靠巴结，是靠信息优势。他掌握了罗德里克等人的囤积账册，用情报交换政治资本。但他也在被这个体制改变——他开始习惯了用钱和关系解决问题 |
| **红月日** | 当预言说王都将在78天后毁灭——一个已经在商会体系中拥有了资产和关系的人，是选择带着一切逃亡，还是留下来保护自己建立的东西？莉迪娅的选择是什么——她的家族利益与她自己的道德判断站在两边 |

### 感情线

| 女主 | 说明 |
|---|---|
| **莉迪娅（商人贵族之女）**——主 | 商业对等→感情。最"成人"的感情线——不是因为年龄，而是因为两人对感情的态度都极度克制和实用主义。他们不是"坠入"爱河——他们是"算出"对方是自己能找到的最好的人，然后等感情追上这个计算 |

### 本线独有的叙事张力

- **阶级的不可跨越性**：无论男主在商业上多成功，他是异世界人、无身份、无神力——在莉迪娅的家族眼中永远不够格。这是"灰姑娘"的性别反转版本
- **被体制改变的风险**：男主在正史线中是"体制外逃亡求生"。在本线中他是"体制内博弈"。问题是：博弈太久，你会不会变成体制
- **资本 vs 人性**：当末日来临——粮食囤积可以让你暴富，但也会让更多人饿死。男主怎么选？莉迪娅怎么选？他们是商人——但他们也是人

## IF-E线：精灵邂逅路线——「北方的森林在哭，你没听见吗」

### 分歧点

男主拿到120银币后，没有留在王都。他用全部积蓄买了一匹马和补给，**独自向北**——不是因为勇敢，是因为他认为王都太危险（暴乱），北方虽然冷，但可能更安静。

> 他不知道北方是魔王力量的前线。

### 故事走向

| 阶段 | 内容 |
|---|---|
| **北行** | 男主穿越北方边境，沿路见到被废弃的村庄、死去的牲畜、反常的寂静。他在一座废弃的亚人聚落中找到了一份古老地图——上面标注了更北方的东西。出于对"还有什么可以用的"的本能，他继续向北 |
| **遇到伊丝琳** | 在叹息之墙以北的魔潮污染区，他遇到了一个独自穿行在枯死森林中的女性——精灵流亡者伊丝琳（见 `女性角色与感情线.md`）。她正在寻找一种据说可以在魔潮中存活的植物——不是为自己，而是为了苍翠之痕中最后的同胞。她发现这个人类男性独自走在魔潮区——"你是傻还是想死？" |
| **不可能的联盟** | 一个无能力的人类和一个被灭族的精灵——在魔潮前锋线上，他们是彼此唯一的盟友。伊丝琳需要男主帮助她穿越人类控制区（她作为精灵，不敢进入人类领地）。男主需要伊丝琳的知识（魔潮的规律、魔化生物的弱点、在污染区中存活的方法） |
| **苍翠之痕** | 伊丝琳带男主去了苍翠之痕——精灵最后的家园。这里没有他想象中的精灵王国——只有数百个在枯死森林中苟延残喘的幸存者，其中大半是老弱。他们看着男主的眼神不是仇恨——是空洞。他们的世界已经被人类烧掉一次，现在正在被魔潮吞噬第二次 |
| **魔王的秘密** | 精灵长老——一位300多岁、已经接近完全盲目的女性——告诉男主一些事情。关于源流。关于女神和魔王的关系。关于召唤仪式的真正代价。关于"红月"到底意味着什么。这些信息将彻底改变他对这个世界的理解——如果他相信一个垂死的精灵的话 |
| **红月的选择** | 78天后，苍翠之痕将被魔潮完全吞噬——这是精灵长老的计算。她要男主带伊丝琳离开——"我们这些老的留下来就行。这棵树在倒下之前，还能为你们挡一阵。"伊丝琳要走还是留？男主要带她走还是尊重她的选择？ |

### 感情线

| 女主 | 说明 |
|---|---|
| **伊丝琳（精灵流亡者）**——主 | 种族仇恨 → 被迫合作 → "你是例外"的困难认知 → 在种族灭亡的背景下，一个人和一个精灵之间能否有未来 |

### 本线独有的叙事张力

- **最深的世界观揭示**：精灵长老的知识是最接近"真相"的信息源。本线比其他任何线都更早揭示神殿的秘密和魔王的本质——但代价是男主必须在魔潮前锋线上生存
- **存在性的孤独**：伊丝琳不是失去家人——是失去**种族**。她的孤独在这个世界中没有解药。男主能做的不是"治愈她"，而是"陪她一起面对没有解药的孤独"
- **苍翠之痕的命运**：一个正在死去的家园。留下来意味着与家园一起死。离开意味着永远失去它。这是伊丝琳的核心选择——也是感情线的核心张力

## IF线的写作原则

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| IF线写成"正史线的低配版" | 每条IF线应有独立的叙事张力、价值观和感情模式 |
