# 末日倒计时

## 倒计时机制

红月每78天重合一次。每一次都比上一次更红。按当前红色加深速度估算——约**720天**（约两年，约9-10次重合）后，红月将达到预言的"彻底染血"临界点。

> ⚠️ 这不是精确的720天定时器。每一次重合都可能提前触发预言。不确定性本身就是压力。

```
时间线：
                过去                        现在                        未来
    ←────────────────────────┼─────────────────────────────────────→
                             │
    历次重合（红色逐次加深）   │  剩余约720天（约9-10次重合）
                             │
    ────────────────────────●──────────────────────────────────────►
                             │
    约半年前：召唤事件        │  ▲ 每次78天重合都是一次红色警报
    约数月前：沈清岚逃亡      │  ▲ 红月在变红——你能看到它变红
    约数周前：王都暴乱与撤离  │  ▲ 约720天后："彻底染血"临界点
    现在：灰丘建设开始        │
```

## 分阶段倒计时

| 阶段 | 大致剩余时间 | 重合次数 | 核心任务 |
|---|---|---|---|
| **生存期**（当前） | ~700天 | 8-9次 | 解决粮食、建立防御、巩固灰丘 |
| **稳定期** | ~500天 | 6-7次 | 扩大据点、接触关键NPC（许安然等）、解决罗德里克 |
| **发展期** | ~300天 | 3-4次 | 灰丘→领地的质变、救援同学、建立武装 |
| **危机期** | ~150天 | 1-2次 | 国家间政治博弈、神殿与王室的矛盾激化、大规模难民潮 |
| **红月前夕** | ~30天 | 0次 | 最终决策——撤离王都？保卫灰丘？阻止预言？ |
| **红月之夜** | 0天 | — | 预言应验（或不应验？或被曲解？） |

## 倒计时对剧情的影响

1. **长线紧迫感**：720天足够男主从零建立一方势力，但不允许拖沓。每一次红月都在提醒——**时间在走**。
2. **阶段性高潮**：每78天的红月重合是天然的**剧情节点**——适合作为卷的结尾或重大事件的背景
3. **累积的恐惧**：红月每变红一点，王都的恐慌就升级一点。这种恐慌的累积是政治动荡的催化剂
4. **不确定性本身就是叙事资源**：不是"720天后一定出事"而是"可能提前"——让每一次红月重合都带有悬念

## 可能的红月日走向（仅供规划参考）

| 可能性 | 说明 |
|---|---|
| **预言应验** | 王都确实被毁灭——可能是物理性的（魔潮淹没城市），也可能是政治性的（神殿崩溃/国王被推翻/内战中毁灭） |
| **预言被阻止** | 男主等人的行动发现了红月的本质——并采取了某种方式延缓或阻止"彻底染血" |
| **预言被曲解** | "王都毁灭"有完全不同的含义——比如"旧王都的政权被推翻"而非"城市被物理摧毁" |
| **预言是触发而非终点** | 红月之夜是某个更大事件的开端——王都毁灭只是连锁反应的第一环 |
| **不止是王都** | 预言只说了"王都毁灭"——但红月之夜可能影响整个大陆，所有国家都会受灾 |

> ⚠️ 当前不应过早确定红月日的具体走向。保持多种可能性，在剧情推进中自然收敛。

| 国家 | 态度 | 行为 |
|---|---|---|
| **圣光王国** | 神殿利用预言强化控制；王室秘密研究天文数据，试图独立验证 | 表面接受神殿叙事，暗中有不同声音 |

| **北境诸领** | 已经不需要预言——他们每天都在经历末日 | 幸存者关注的是"下一次魔潮什么时候来"，而不是"红月什么时候变红" |
