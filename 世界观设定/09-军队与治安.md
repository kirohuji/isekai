# 军队、治安与武装力量

---

## 一、武装力量总览

| 力量 | 隶属 | 规模 | 主要职能 |
|---|---|---|---|
| **王国正规军** | 王室 | 数万人（主力在北方前线） | 边境防御、大规模作战 |
| **骑士阶级** | 国王册封/贵族 | 数千人 | 重装骑兵、军队指挥层、领主私兵核心 |
| **王国卫兵** | 王室（名义上） | 数千人 | 城市治安、城门、暴乱镇压 |
| **神殿守卫（灰袍）** | 第四圣殿 | 数百人 | 异端追捕、逃犯追踪、神殿保护 |
| **领主私兵** | 地方贵族 | 各数百至数千 | 领地防御、征税执法 |
| **雇佣兵团** | 独立/铁棘公国 | 各数十至数千 | 有偿作战、商队护卫、矿区镇压 |
| **冒险者** | 公会（独立） | 数千人（注册） | 委托执行、魔物讨伐、半军事化战力 |

## 二、王国正规军

| 维度 | 内容 |
|---|---|
| **总部** | 王都外军营 |
| **当前部署** | 主力在北方叹息之墙。王都留守不足三成——暴乱未被迅速镇压的原因 |
| **兵源** | 强制征兵（贫民）、志愿兵（佣兵/老兵）、骑士（贵族子弟充任军官） |
| **装备** | 步兵：皮甲/铁甲、长矛、短剑、盾。骑兵：骑士阶级充任，重装 |
| **与神殿的关系** | 随军祭司名义提供祝福，实际监控军队动向 |

### 强制征兵

王国法律允许在战时或戒严期间强制征召成年男性。征兵官优先选择贫民——他们没有钱贿赂征兵官。被征召者需自备口粮。逃兵役者全家受罚。

## 三、骑士阶级

骑士是贵族军事体系的底层，也是军队的指挥骨干。在中世纪式的封建体系下，骑士既是战斗单位，也是社会身份。

| 维度 | 说明 |
|---|---|
| **受封** | 由国王或高阶贵族册封。途径：战功、长期服役、或向王室捐赠大笔金钱 |
| **军事角色** | 重装骑兵——军队中最具冲击力的兵种。骑士本人需自备战马、盔甲和武器（极其昂贵） |
| **经济负担** | 一套完整的骑士装备（战马+全身甲+长剑+ lance）约等于一个中等商人家庭数年的收入。许多骑士家族因此负债 |
| **封地骑士** | 有封地。从封地税收中维持武备。暴乱后封地收入下降→部分骑士破产 |
| **无地骑士** | 无封地，靠薪俸或佣兵报酬维生。通常是骑士家族的次子或破产骑士 |
| **与公会的关系** | 骑士是公会赞助人的主力阶层。退役骑士常投资公会或担任冒险队的战斗顾问 |
| **当前困境** | 物价上涨、封地收入下降、装备维护成本高昂。越来越多骑士转向佣兵或依附大商人 |

### 骑士的日常

骑士并非每天都在战场。大部分时间在管理封地、训练、或在贵族府邸担任护卫队长。战时被征召。和平时期最怕的不是敌人——是穷。战马每天吃掉的草料价值抵过一个贫民家庭的三餐。

## 四、王国卫兵

| 维度 | 内容 |
|---|---|
| **隶属** | 王室（名义上）。实际指挥权被神殿和地方贵族瓜分 |
| **职责** | 城门守卫、街道巡逻、集市秩序、暴乱镇压 |
| **装备** | 皮甲、短剑、木盾。军官配铁甲和长剑 |
| **驻扎** | 城门塔楼、南区军营。北区贫民窟无固定驻守 |
| **腐败** | 基层卫兵主要收入来自贿赂。"保护费"是公开的秘密。暴乱时欠薪数月——卫兵消极应对 |
| **暴乱后** | 人数翻倍。每个路口设岗。宵禁——日落至日出禁止上街 |

## 五、神殿守卫（灰袍）

| 维度 | 内容 |
|---|---|
| **隶属** | 第四圣殿（异端审判与逃犯追捕） |
| **职责** | 追踪异世界逃犯、保护神殿财产、执行异端搜查 |
| **装备** | 灰袍（便于城市隐蔽）、细剑、异世界检测铃 |
| **特殊权力** | 可绕过卫兵直接搜查住宅。可扣押任何人进行"信仰核验" |
| **与卫兵关系** | 灰袍权力高于卫兵，但人数远少。日常治安是卫兵的活——灰袍只在"神殿事务"上介入 |

## 六、领主私兵

每个伯爵以上的贵族都维持自己的私人武装。这在中世纪式封建体系下是常态——国王的军队本质上是各领主私兵的联合体。

| 维度 | 说明 |
|---|---|
| **规模** | 边境伯爵：数百至一千。公爵：可达数千 |
| **兵源** | 领地内征召的自由民+雇佣的职业士兵+骑士及其侍从 |
| **指挥权** | 完全属于领主。国王无权直接指挥——需要领主同意 |
| **与王权的关系** | 领主私兵是国王最头痛的问题。国王需要领主的兵来打仗，但领主的兵越多，国王的权力越弱 |
| **在暴乱中** | 大部分领主私兵按兵不动——保护自家围墙，而非镇压街头暴乱 |

## 七、雇佣兵团

雇佣兵是这个世界的**影子军队**。他们是专业的战斗者，不属于任何国家、任何领主——只为出价最高的人服务。

| 维度 | 说明 |
|---|---|
| **来源** | 退役士兵、破产骑士、无地农民、逃亡奴隶、冒险者转业 |
| **组织方式** | 以团为单位。规模从数十人到数千人不等。团长通常是退役的高级骑士或资深佣兵 |
| **主要雇主** | 商人（商队护卫）、贵族（私兵补充）、神殿（矿区镇压）、公会（高危委托） |
| **铁棘公国的佣兵产业** | 铁棘是大陆最大的佣兵输出国。佣兵训练营是铁棘的重要产业——训练→出口→赚外汇。铁棘佣兵以重装步兵闻名 |
| **忠诚度** | 对合同忠诚，而非对雇主。合同到期，佣兵可以转身加入昨天的敌人。这是佣兵的行规——不是背叛，是生意 |
| **哈罗德的佣兵经历** | 退伍后给商人当过护卫，替商队杀过人。他认识不少佣兵——通过他们获得了矿区和北境的情报 |

### 佣兵团的等级

| 等级 | 规模 | 声誉 | 收费 | 典型雇主 |
|---|---|---|---|---|
| **金牌佣兵团** | 500-2000人 | 大陆知名。数十年历史 | 极高。按战役计价 | 国家、大贵族 |
| **银牌佣兵团** | 100-500人 | 区域内知名 | 高。按合同计价 | 地方贵族、大商人 |
| **铜牌佣兵团** | 20-100人 | 本地运营 | 中。按天或按任务 | 商队、小镇 |
| **散兵** | 个人或三五人 | 无组织 | 低。按天 | 任何人 |

哈罗德属于散兵——独来独往，接短期护卫。

## 八、冒险者作为准军事力量

冒险者在法律上不是军队，但在功能上——特别是在偏远地区——他们是事实上的武装力量。

| 维度 | 说明 |
|---|---|
| **战斗能力** | 中高级冒险者的战斗经验不亚于正规军。区别在于：军人打阵地战，冒险者打遭遇战 |
| **公会委托中的军事任务** | 讨伐魔物、清剿盗匪、护送要人——本质上都是军事行动 |
| **与领主私兵的关系** | 领主常通过公会雇佣冒险者来补充私兵——比自己养兵便宜 |
| **在末日中的角色** | 如果王国军队崩溃，冒险者公会可能成为唯一有组织的武装力量 |

## 九、暴乱后的戒严体制

临时戒严令：
- 成年男性须登记，可被征召为临时民兵
- 粮仓、药铺、铁匠铺列为"战略资源单位"，接受定期检查
- 未经许可不得囤积超过十日消耗量的粮食
- 违反者当场拘押，不需审判

这些措施给了卫兵和灰袍极大的自由裁量权。

## 十、哈罗德：一个退伍兵的轨迹

哈罗德的经历是王国底层士兵的缩影：

军队服役（北方边境，失右眼）→ 无抚恤退伍 → 佣兵散活（商队护卫）→ 旅馆守夜（稳定但低薪）→ 重返佣兵（矿区护送，三倍工钱）

他认识的人：军队底层士兵、卫兵小队长旧战友、商队的佣兵同僚。这些关系网是旅馆情报来源之一——提前知道哪条街有突击检查、哪条路被封。

## 十一、与故事的关联

- 暴乱后高压管控直接推动男主离开王都
- 灰袍与卫兵的权力重叠制造了可利用的灰色地带
- 哈罗德的军队-佣兵双轨经历是旅馆情报源
- 铁棘佣兵产业是未来可能的武力来源——如果男主有钱雇
- 骑士阶级的困境（封地收入下降）意味着更多骑士可能转向冒险者公会——公会的力量在增长
