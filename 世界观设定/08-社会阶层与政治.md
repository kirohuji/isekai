## 一、阶层金字塔

| 阶层 | 构成 | 占比 | 说明 |
|---|---|---|---|
| **王室** | 国王、王族 | <0.1% | 名义最高。实权被神殿和贵族瓜分 |
| **高阶贵族** | 公爵、侯爵、伯爵 | <1% | 领地统治。私兵。与神殿博弈 |
| **神殿高层** | 大神官、枢机祭司 | <0.1% | 精神与司法权威。实际权力可能超过国王 |
| **骑士阶级** | 受封骑士、骑士家族 | ~2% | 军事贵族底层。有封地或薪俸。公会赞助人的主力 |
| **大商人** | 行会头领、赞助人级富商 | ~1% | 无爵位，财富和影响力不亚于贵族。罗德里克属于此层 |
| **中低阶贵族** | 子爵、男爵、下级神职 | ~3% | 依附上层。管理具体事务 |
| **中小商人/工匠** | 行会成员、铁匠、药师 | ~15% | 有稳定收入。一两次厄运就可能滑落 |
| **自由平民** | 农夫、车夫、摊贩、佣兵 | ~35% | 体力维生。粮价波动决定生死 |
| **贫民** | 短工、拾荒者、乞丐 | ~30% | 北区贫民窟。暴乱主力。寿命<40 |
| **奴隶** | 战俘、债务奴、买卖奴 | ~15% | 不被视为人。灰绳市场是最大交易所 |

## 二、王室

| 维度 | 内容 |
|---|---|
| 国王 | 约五十岁。名义最高统治者。政令需神殿盖章生效 |
| 当前处境 | 暴乱后神殿要求更多权力，地方贵族要求更多自治。两面受压 |
| 对召唤者的态度 | 可能私下同情。曾反对女性召唤者"保护"制度，但无力改变 |
| 王族 | 王后已故。太子驻北方边境，次子在王都极少露面 |

## 三、贵族体系

| 爵位 | 权力范围 | 代表人物 |
|---|---|---|
| 公爵 | 管辖一省 | 尚未出场 |
| 侯爵 | 管辖数城 | 尚未出场 |
| 伯爵 | 管辖一城及周边 | 暮河镇边境伯爵、铁脊伯爵（矿区） |
| 子爵 | 管辖一镇 | — |
| 男爵 | 管辖数村或王都内一区 | — |

大多数贵族住在王都南区高墙宅邸内。暴乱中损失微乎其微——南区围墙挡住了暴民。部分贵族趁机低价收购被烧毁的商铺和土地。

## 四、骑士阶级

骑士是贵族体系的底层，也是军事力量的中坚。不同于高阶贵族（靠领地收入维生），骑士阶层更依赖**薪俸、战利品和赞助**。

| 维度 | 说明 |
|---|---|
| **受封方式** | 由国王或高阶贵族册封。战功、服役、或金钱捐赠均可获封 |
| **经济来源** | 封地收入（小骑士）、王室/贵族薪俸、佣兵报酬、公会赞助人收益 |
| **与公会的关系** | 骑士阶级是冒险者公会**赞助人的主力**。退役骑士或骑士家族次子常投资公会：出钱委托→获得情报和精锐冒险者的优先使用权 |
| **社会地位** | 高于富商但低于伯爵。有佩剑权和纹章。平民需对骑士行礼 |
| **当前困境** | 暴乱后封地收入下降，物价上涨。部分骑士转向佣兵或投靠大商人 |

## 五、大商人

大商人是没有爵位的实际权力者。他们的财富和影响力往往超过中小贵族。

| 维度 | 说明 |
|---|---|
| **财富来源** | 贸易垄断、矿产、药材、粮食囤积、奴隶交易 |
| **权力基础** | 不是血统——是钱。通过**贿赂、联姻和赞助**渗透贵族和神殿 |
| **与公会的关系** | 大商人是冒险者公会的**核心赞助人**（白银/黄金级）。通过赞助获取：情报（冒险者遍布全国）、物流优先权、政治筹码 |
| **与神殿的关系** | 既合作又博弈。罗德里克行贿第六圣殿地方官。神殿需要商人的物资和运输网络 |
| **代表人物** | **罗德里克**（暮河镇药材商）——拖欠亚人货款、举报逃奴、囤积粮药。铁棘公国的武器出口商。苍海联邦的盐商 |

**大商人 vs 骑士阶级**：骑士有地位无现金。大商人有现金无地位。两者经常合作——骑士需要赞助来维持武装，商人需要骑士的武力保护商队。公会是两者最常交汇的场所。

## 七、地方权力

### 边境伯爵

王室任命，实际独立运作。拥有私兵、征税权和审判权。暮河镇的边境伯爵与第六圣殿和罗德里克三方共存。

### 罗德里克

大商人阶层的缩影。发家路径：拖欠亚人货款→举报逃奴→低价收购银叶草田→行贿税务官→囤积粮药→暴乱后高价出售。

## 八、政治派系

| 派系 | 代表 | 立场 |
|---|---|---|
| **神殿派** | 神职人员、依附神殿的勇者、神殿供养的商人 | 支持神殿扩权 |
| **王权派** | 传统贵族、王室亲属、部分军队将领 | 支持国王收回权力 |
| **地方自治派** | 边境伯爵、偏远领地贵族 | 反对中央集权 |
| **骑士-商人联盟** | 骑士阶级+大商人+公会赞助人网络 | 不站队，但在公会体系内形成跨阶层利益共同体 |
| **中立观望派** | 大部分中小贵族和商人 | 等局势明朗 |

## 九、贫民区与地下世界

### 贫民窟（北区）

木板泥砖棚屋。6-8人挤一间。无卫兵巡逻。纠纷靠拳头。死人不是新闻。疾病：伤寒、肺病、营养不良。费恩偷药渣给贫民治病。

### 灰手盗贼行会

松散犯罪网络。无总部，无正式头目。销赃、走私、情报买卖、高利贷。莱恩曾在灰手外围跑腿。

### 奴隶市场与娼馆

灰绳市场：每周开市两次。廉价娼馆（5铜币/次）到高级娼馆（银币计价）。神殿庄园：女性无能力召唤者的性奴隶集中营。

### 暗杀

贵族和商人之间不罕见。毒药最常用——不留血、不出声。

## 十、阶级压迫的日常

- 奴隶上街须带契约牌。无牌可当场处决或转卖
- 贫民进入商业区可被卫兵以"看你像小偷"为由盘查
- 贵族犯罪几乎不受追究。打死平民按"财产损失"赔钱
- 亚人在王都内行动受限。多数无法获得城内居住许可
