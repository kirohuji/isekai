# 诸国与大陆

## 大陆总览

大陆名为**埃瑟兰**（Aesland）。已知的人类文明分布在**五个主要势力区域**中。这五个势力并非全部对等——有些是统一王国，有些是松散联盟，有些已经名存实亡。

圣光王国（故事当前舞台）位于大陆中部偏南。它是五个势力中**神殿影响力最强**的一个——也是目前唯一在积极应对魔王威胁的一个。

```
                        北 方 荒 野（魔族领地）
                                │
                    ┌───────────┼───────────┐

                    │ （半沦陷区）  │           │

                            │               │
        铁棘公国 ───── 圣光王国 ───── 东方帝国
        （西部）      （中部·主舞台）    （东部·闭关）
                            │
                    苍海联邦（南部）
```

## 一、圣光王国（Holy Light Kingdom）——当前主舞台

| 维度 | 内容 |
|---|---|
| **正式名称** | 圣光王国 |
| **首都** | 王都（正式名称待定） |
| **政体** | 封建神权王国。国王 + 神殿共治，神殿在宗教和司法上占优 |
| **当前君主** | 国王——姓名待定。神殿的大神官实际权力可能超过国王 |
| **人口** | 约200-300万人 |
| **领土** | 大陆中部和南部——从北方的叹息之墙到南方的河运平原 |
| **经济支柱** | 农业（南方河运区）、矿产（西部矿区——但矿区与铁棘公国有领土争议）、贸易（王都商业区、暮河镇等边境贸易点） |
| **军事** | 王国正规军 + 神殿骑士团 + 勇者体系。目前大量军队被牵制在北方叹息之墙、西部矿区镇压和内部维稳 |

### 当前困境

| 问题 | 说明 |
|---|---|
| **魔王威胁** | 北方叹息之墙守备严重不足。但王国将大量资源用于内部维稳（镇压暴乱、追捕逃奴）而非前线防御 |
| **神殿与王室的内斗** | 国王可能正在暗中收集神殿隐瞒真相的证据。神殿则在利用战时状态扩大自身权力 |
| **地方离心力** | 暮河镇的边境伯爵体系实际上半独立。地方权力与商人（如罗德里克）勾结腐败，对王都的命令阳奉阴违 |
| **战时经济崩溃** | 暴乱频发、粮价暴涨、黑市猖獗。王国正在向"高压维稳→更多暴乱→更高压"的恶性循环滑落 |
| **铁棘公国的领土压力** | 西部矿区是圣光王国与铁棘公国的争议领土。铁棘公国可能利用圣光王国的困境提出领土要求 |

### 分裂的可能性

> 🔴 **圣光王国不是铁板一块。** 在末日压力下，边境伯爵、地方大贵族、甚至某些勇者集团都可能选择**独立或投靠其他国家**。国王的统治从来不是绝对的——而是靠资源分配和神殿背书维持的。当资源枯竭、神殿的叙事漏洞越来越多——**这个王国可能分裂**。

## 二、铁棘公国（Iron Thorn Duchy）——西部邻居

| 维度 | 内容 |
|---|---|
| **正式名称** | 铁棘公国 |
| **首都** | 铁棘堡（Iron Thorn Keep） |
| **政体** | 军事贵族寡头制。公爵是最高统治者，但实权由多个军事贵族家族分享。**神殿在此国的影响力远弱于圣光王国**——铁棘有自己的军神信仰传统，对"女神"信仰持实用主义态度（用得着的时候拜一下，用不着的时候不理会） |
| **人口** | 约100-150万人 |
| **领土** | 大陆西部——丘陵、山地、部分矿区 |
| **经济支柱** | 矿产（铁、铜、银）、武器锻造、雇佣军输出 |
| **军事** | 以重装步兵闻名。铁棘的军事贵族从小接受战斗训练。此外有大量佣兵团在此注册和训练 |

### 与圣光王国的关系

| 维度 | 说明 |
|---|---|
| **领土争议** | 铁脊矿区（西部最大的矿区）——两国都声称主权。目前实际由圣光王国控制，但铁棘公国一直在寻找夺回的机会 |
| **贸易依赖** | 铁棘需要圣光王国的粮食（铁棘领土多山，耕地不足）。圣光需要铁棘的矿石和武器。双方在贸易上互相需要——这暂时抑制了军事冲突 |
| **神殿的敌意** | 铁棘对神殿的不敬在圣光王国高层眼中是"准异端"。神殿曾试图在铁棘建立圣殿分支——遭到军事贵族的集体抵制 |
| **末日中的立场** | 铁棘尚未受到魔潮的直接影响——他们在圣光王国的西侧，魔王力量从北方来，暂时被圣光王国"挡住"。这使得铁棘的立场暧昧——**是帮助圣光对抗魔王，还是趁圣光衰落时夺取领土？** |

### 铁棘公国的阴暗面

- 奴隶制比圣光王国更加残酷——矮人奴隶是矿产经济的基础
- 佣兵团输出是重要产业——铁棘训练佣兵然后卖给其他国家打仗
- 军事贵族内部血腥的权力斗争——公爵的位子是杀上去的，不是继承的

## 三、北境诸领（Northern Territories）——前线半沦陷区

| 维度 | 内容 |
|---|---|
| **正式名称** | 无统一名称。通常被称为"北境"或"北方领" |
| **政体** | 名义上是圣光王国的北方边境领地，实际上**已经在崩溃**。由十几个半独立的领地组成，每个由一位领主（或领主死后由其幸存的继承人/军队指挥官）控制 |
| **首都** | 无。最大的定居点是**叹息之墙**上的要塞群 |
| **人口** | 已难以统计。大量居民已南逃。剩余人口可能不足十万 |
| **领土** | 圣光王国北方——从北部边境到叹息之墙 |
| **现状** | 🔴 **正在沦陷。** 多个村庄和亚人聚落已完全失联。叹息之墙的守备严重不足（原两千驻军→现可能不足三百）。北境诸领实际上已经是"幸存者孤岛"的集合——而不是一个能组织有效防御的行政实体 |

### 与男主阵营的潜在关联

- 失联村庄中可能有男主需要的资源或情报
- 北境难民的南迁可能波及东南沼泽——带来人口压力、疾病传播和情报
- 如果叹息之墙失守，圣光王国的整个北方将暴露在魔族面前——这可能会**加速各国的政治博弈**

## 四、苍海联邦（Azure Sea Federation）——南方海上贸易联盟

| 维度 | 内容 |
|---|---|
| **正式名称** | 苍海联邦 |
| **政体** | 城邦联盟。由七个沿海城邦组成的松散贸易同盟。无统一君主——最高决策机构是"七城议事会"，由各城邦的执政官组成 |
| **人口** | 约150-200万人 |
| **领土** | 大陆南部海岸线和近海岛屿 |
| **经济支柱** | 海上贸易、造船、渔业、盐业。是大陆最富庶的地区 |
| **军事** | 海军强大（大陆唯一成规模的海军）。陆军薄弱——依赖雇佣兵（经常从铁棘公国购买佣兵服务） |
| **神殿态度** | 联邦宪法保证"信仰自由"——这实际上是限制神殿权力。神殿在联邦有圣殿分支，但权力远不如在圣光王国 |

### 与圣光王国的关系

| 维度 | 说明 |
|---|---|
| **贸易伙伴** | 圣光王国的粮食出口到联邦，联邦的海产品和盐出口到圣光。双方经济上互补 |
| **魔王威胁的态度** | 🔴 **苍海联邦不相信魔王威胁。** 联邦的执政官认为"魔王危机"是神殿编造的——目的是扩大神殿权力和征收更多税。联邦拒绝向圣光王国提供军事援助对抗魔王 |
| **末日中的风险** | 如果魔王被证明属实——联邦将处于极度不利的地位。海军对陆地上的魔族毫无作用。联邦的城墙从未被攻击过——他们不知道如何防御 |

### 苍海联邦的特色

- 奴隶制是非法的——但"契约劳工"制度本质上就是奴隶制的改名
- 亚人在联邦的地位略高于圣光王国（部分城邦甚至允许亚人拥有财产）——但仍然是二等居民
- 存在一个**秘密的学术团体**（自然哲学学会），研究天文、医学和古代文献——不受神殿审查。这可能是一个未来的情报来源

## 五、东方帝国（Eastern Empire）——闭关的古国

| 维度 | 内容 |
|---|---|
| **正式名称** | 旭日帝国（Empire of the Rising Sun） |
| **首都** | 天阳城 |
| **政体** | 绝对君主制——皇帝拥有至高无上的权力。**没有神殿体系。** 帝国有自己的国教——"天道信仰"（崇拜天、地和祖先，不崇拜人格神）。女神信仰在帝国被视为"蛮族迷信" |
| **人口** | 推测500万人以上——大陆人口最多的国家 |
| **领土** | 大陆东部，被"龙骨山脉"与圣光王国隔开。只有少数山口通道可供通行 |
| **闭关政策** | 帝国实行严格的闭关政策——不允许外国人随意入境，不允许本国人随意出境。与圣光王国的外交关系仅维持最低限度的使节交换 |
| **魔王威胁** | 帝国知道魔王的存在——但不认为魔王能越过龙骨山脉。帝国的态度是："这是你们西方人的问题" |

### 与圣光王国的关系

| 维度 | 说明 |
|---|---|
| **历史** | 两国在历史上发生过两次战争——均因圣光王国试图向帝国传教（派遣神殿传教士） |
| **当前** | 冷战式共存。互不信任，互不侵犯。贸易量极小 |
| **末日的变数** | 如果魔王力量扩散至整个大陆——龙骨山脉也无法阻挡魔潮。帝国能否在末日中独善其身——还是被迫与圣光王国合作？ |

## 六、男主建立领地的可能性

### 从灰丘到一方势力

当前灰丘是一块荒废高地 + 4个人 + 零粮食 + 零防御。但如果给男主**两年的时间**（而非78天）——从灰丘发展到被周边势力承认的一方领地，是合理的发展弧线。

| 发展阶段 | 条件 | 时间估算 |
|---|---|---|
| **生存期**（当前） | 粮食≥7天、基本防御、药品储备 | 1-2个月 |
| **稳定期** | 粮食自给、防御完整、收入来源、常驻人口5-15人 | 3-6个月 |
| **聚落期** | 常驻20+人、基础设施完善、与苇水村形成经济互补、被周边承认 | 6-12个月 |
| **领地期** | 有自己的武装（民兵/猎人网络）、被暮河镇等地方势力承认、开始接收逃难者 | 12-18个月 |
| **势力期** | 在王国政治版图上有一席之地——可能被王室或神殿正式册封（或通过事实上的控制力获得承认）。开始救援同学、与亚人网络深度融合 | 18-24个月 |

### 领地崛起的关键资源

| 资源 | 来源 | 获取难度 |
|---|---|---|
| **人口** | 逃难者（北境难民、矿区逃奴、被追捕的亚人） | 🟡 接收难民会带来粮食压力和安全风险 |
| **武力** | 亚人猎人网络（格兰等）、逃亡矮人铁匠（锻造武装）、可能加入的佣兵 | 🟡 需要信任和资源交换 |
| **合法性** | 神殿的册封、国王的承认、或"事实上的控制" | 🔴 极难。但末日中——合法性可能可以用粮食和安全来换 |
| **情报** | 沈清岚的名单、亚人网络、鼠族商队 | 🟡 已有基础 |
| **经济** | 药草贸易（银叶草等）、走私网络、矮人锻造品 | 🟡 需要建设生产和贸易渠道 |

### 领地的政治定位

男主的领地在多个势力之间——这既是危险也是机会：

| 势力 | 对男主领地的可能态度 |
|---|---|
| **圣光王国** | 初期无视。如果领地壮大→要么册封收编，要么镇压 |
| **暮河镇（边境伯爵）** | 竞争关系。领地可能吸走暮河镇的人口和贸易 |
| **铁棘公国** | 可能暗中支持——作为牵制圣光王国的棋子 |
| **苍海联邦** | 可能的贸易伙伴——如果领地能产出有价值的商品 |
| **亚人网络** | 最可能的长期盟友——如果男主证明自己与"其他人类"不同 |
| **神殿** | "持续观察"可能升级为"必须控制或消灭" |

## 七、设定原则

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| 把其他国家写成背景板 | 每个国家应有独立的利益、动机和内部矛盾 |
| 领地发展过快 | 灰丘→领地的过程应充满挫折、反复和代价 |
| 男主靠"个人魅力"获得政治承认 | 末日中的政治承认基于：你能提供什么（安全、粮食、贸易）以及你能威胁什么 |
| 把神殿和王室当成唯一的外部压力 | 邻国、商人公会、佣兵团——都是可能的盟友或敌人 |
