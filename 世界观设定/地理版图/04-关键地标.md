# 关键地标

## 说明

本文档收录**超出男主当前活动范围**的重要地标。男主目前已到达的地点（灰丘、苇水村、暮河镇、东南沼泽等）见 `地理版图/06-地理全图.md`。

---

## 一、王都核心地标

### 大圣堂（第一圣殿）

| 维度 | 内容 |
|---|---|
| **位置** | 王都中心，神殿区最高处 |
| **外观** | 白石建造的巨大圆顶建筑。顶部设有发光水晶尖塔——神殿宣称这是女神祝福的具象化，夜里在王都任何角落都能看到其光芒 |
| **功能** | 大神官驻地、最高级宗教仪式场所、核心勇者初次祝福仪式的举行地 |
| **禁区** | 核心区域只有枢机祭司以上级别可进入 |
| **叙事意义** | 神殿权力的物质象征。如果有一天大圣堂的光芒熄灭——王都的信仰秩序将崩塌 |

### 第三圣殿档案馆

| 维度 | 内容 |
|---|---|
| **位置** | 王都神殿区，与第一圣殿相邻 |
| **外观** | 灰色石质建筑，窗户极少，内部恒温恒湿——为保存文件而设计 |
| **收藏内容** | 历次召唤的完整记录（可证明召唤并非第一次）；五十名同学的能力鉴定和分配记录；神殿数百年来的财务和物资账册；被销毁或隐藏的预言记录副本 |
| **安保** | 神殿内部区域。只有第三圣殿的神职人员可以进入 |
| **叙事意义** | **沈清岚曾在此抄写。** 这里存放着神殿最深层的秘密。如果男主阵营未来能以某种方式获取其中的完整记录，将揭示一切真相 |

### 灰绳市场

已设定。见 `地理版图/06-地理全图.md`。王都合法与非法的奴隶交易中心。

## 二、北方的关键地标

### 叹息之墙（Wall of Sighs）

| 维度 | 内容 |
|---|---|
| **位置** | 北方边境与北部荒野的分界线 |
| **建造年代** | 数百年前。建造者的身份已遗失——神殿声称是"女神引导下的先王"所建，亚人传说则认为建造者是"上一个被魔王毁灭的文明" |
| **规模** | 延绵数十里的石质防御工事，配有多处堡垒和瞭望塔。最高处约十五米 |
| **原驻军** | 约两千人 |
| **当前驻军** | 🔴 推测不足三百人（守备被抽调用于王都维稳） |
| **叙事意义** | 王国对抗魔王的第一道——也可能是最后一道——物理防线。如果叹息之墙失守，魔王力量的扩散速度将急剧加快 |

### 古代迷宫（Great Maze）

| 维度 | 内容 |
|---|---|
| **位置** | 叹息之墙以北的荒野中。确切位置无人知晓——曾有探险者声称找到了入口，但无人返回提供可靠地图 |
| **性质** | 远古文明的巨大建筑群遗迹。地表以上是残破的石柱和崩塌的穹顶；地表以下据说有多层地下结构——通道、大厅、密室 |
| **神殿的说法** | 女神封印魔王第一具"肉身"的地点，凡人不得进入 |
| **亚人的说法** | 前代文明的遗产，其中藏有关于源流、女神和魔王的真相 |
| **叙事意义** | 可能是揭示一切真相的终极场景之一。如果在红月日前需要某个关键物品或知识来阻止预言——古代迷宫是最可能的藏匿地点 |
| **当前可接触性** | 零。78天内根本不可能到达并返回。但其存在可以通过许安然的预言、亚人老祭司的记录或神殿的秘密档案被提及 |

### 亚人故地

| 维度 | 内容 |
|---|---|
| **位置** | 叹息之墙以南，北方边境的森林地带 |
| **性质** | 兽耳族的传统聚居区。曾存在多个大型聚落——有些聚落有上百年历史 |
| **现状** | 🔴 已被魔化力量和人类扩张双重破坏。大部分亚人被迫南下。聚落虽已废弃，但可能遗留：古老记录、药草库存、藏匿的族人（不愿或无法离开的老弱者） |
| **叙事意义** | 格兰和米娅可能知道故地的某些秘密。如果未来有救援或探索任务需要进入北方——故地是必经之地 |

## 三、西部的关键地标

### 铁脊矿区（Iron Ridge Mines）

| 维度 | 内容 |
|---|---|
| **位置** | 西部矿区的核心地带 |
| **性质** | 王国最大的铁矿和银矿来源 |
| **劳动力** | 数千名奴隶、罪犯和被惩罚的召唤者 |
| **管理者** | 铁脊伯爵家族（地方大贵族）+ 神殿派驻的监督祭司 |
| **当前事件** | 陈浩（火系高级能力者）在此镇压逃奴暴动 |
| **叙事意义** | 代表着王国对底层的最残酷剥削。可能有被神殿罚到矿区的同学——这些是沈清岚名单上的潜在救援目标（但距离遥远，救援可行性低） |

## 四、传说中未经证实的地标

以下地标的存在性未在故事中确认。它们来自不同来源的传说，可信度不一。

| 地点 | 来源 | 传说内容 | 可信度 | 叙事潜力 |
|---|---|---|---|---|
| **初光圣殿** | 神殿教典 | 女神最初降临世界的地点。据称隐藏在"凡人不可见之处" | 🟡 中 | 如果真实存在，可能揭示女神真相 |
| **逆流之泉** | 亚人古老传说 | 可以净化魔化污染的泉水。水往高处流——违反物理规律 | 🟡 中（亚人老祭司可能知道更多） | 末日中极为珍贵。净化男主被腐兽抓伤的潜在魔化 |
| **源流祭坛** | 亚人源流信仰 | 远古时代用于"接通源流"的祭坛。据称位于大地裂缝深处 | 🟢 低 | 可能与召唤仪式的秘密有关 |
| **深渊裂隙** | 亚人老祭司的说法 | 魔王力量渗入世界的主要通道。一条深不见底的大地裂缝，从中涌出暗红色的光 | 🟡 中 | 如果真实存在，封印或关闭它可能是最终目标 |

## 六、地标信息在叙事中的释放策略

| 地标 | 男主当前是否知道 | 知道渠道 | 首次可提及的时机 |
|---|---|---|---|
| 大圣堂 | 是（曾在王都生活） | 亲眼见过 | 随时 |
| 第三圣殿档案馆 | 通过沈清岚了解 | 沈清岚的叙述 | 已可提及 |
| 叹息之墙 | 可能知道 | 通用地理知识 | 当北方话题出现时 |
| 古代迷宫 | 不知道 | 亚人老祭司或许安然的预言 | 中后期 |
| 亚人故地 | 通过格兰了解部分 | 格兰的叙述 | 当格兰讲述过去时 |
| 铁脊矿区 | 可能知道 | 通用地理知识 + 陈浩在那里的消息 | 当矿区话题出现时 |
| 传说地标 | 不知道 | 亚人老祭司、神殿档案、许安然 | 中后期 |

## 设定原则

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| 所有地标一次性揭示 | 远处地标通过传闻、信件、预言逐步引入 |
| 把地标写成"副本关卡" | 每个重要地标应有叙事功能——揭示信息、推动关系、制造抉择——而不仅仅是"打怪的地方" |
| 男主随意到达远处地标 | 78天的时间限制 + 资源匮乏 + 周启明封锁 = 活动范围严重受限 |
| 地标之间毫无关联 | 地标之间存在信息链——如：档案馆记录 → 古代迷宫位置 → 深渊裂隙真相 |
