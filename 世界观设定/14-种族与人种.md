# 种族与人种

## 总览

这个世界存在**多个智慧种族**。人类是统治性多数，但并非唯一的智慧存在。

| 种族 | 人口占比 | 社会地位 | 主要分布 |
|---|---|---|---|
| **人类** | ~70% | 统治阶层 | 全境，集中在中部和南部 |
| **亚人（四亚种）** | ~22% | 被压迫/边缘阶层 | 边境、沼泽、森林等边缘地带 |
| **精灵** | ~5% | 濒临灭绝的隐匿种族 | 极北残存森林（苍翠之痕）、少数流亡者散落各地 |
| **矮人** | ~3% | 被奴役的工匠种族 | 西部矿区（被强制劳动）、少数逃奴散落东南边境 |

---

## 一、人类（Human）

### 社会分层

人类的阶层由**神力等级 + 血统 + 财富**三者共同决定（详见 `社会制度/02-社会结构.md`）。

### 对亚人的普遍态度

| 阶层 | 典型态度 |
|---|---|
| **神殿** | 亚人是"未受女神祝福的残缺种族"，不承认其平等地位，但不主动灭绝——因为需要劳动力 |
| **贵族** | 视亚人为可剥削的廉价劳动力和异域商品（有翼族的羽翼是身份象征） |
| **商人** | 视亚人为贸易对象（经常欺诈），也视其为走私竞争对手 |
| **边境平民** | 务实共存（如苇水村）。意识形态让位于生存需要 |
| **王都平民** | 受神殿宣传长年影响，对亚人多有恐惧、蔑视和猎奇心态 |

## 二、亚人（Demi-Human）

亚人是一个**统称**，包含多个生物学上不同的亚种。它们彼此之间通过共同的历史创伤（人类压迫）形成了松散的身份认同——即"亚人网络"——但各亚种的文化和利益并不统一。

### 2.1 兽耳族（Beast-kin）

最普遍的亚人类型，约占亚人总人口的60%。

| 维度 | 内容 |
|---|---|
| **外部特征** | 头顶生长动物耳朵（狼、猫、狐、犬等变体），部分个体有尾巴。除此之外与人类外貌相似 |
| **体型** | 与人类相当，狼系偏高大，猫系偏纤细 |
| **寿命** | 约60-70年（略短于人类） |
| **天生能力** | 感官增强——夜视（弱光环境可辨识轮廓）、远距离听觉、气味追踪。在神殿能力鉴定中常被笼统归为"低级感知系" |
| **代表角色** | **格兰**（狼系，猎人）、**米娅**（灰狼系，格兰之女） |
| **社会角色** | 猎人、向导、药农、边境贸易者 |
| **与人类关系** | 最常见的人类接触亚人类型。边境部分村庄（如苇水村）与兽耳族有共存传统 |

#### 兽耳族的内部变体差异

| 变体 | 体型特征 | 行为倾向 | 常见职业 |
|---|---|---|---|
| **狼系** | 偏高大，肌肉发达 | 重视家族和群体忠诚，集体行动 | 猎人、战士、聚落守卫 |
| **猫系** | 纤细敏捷，夜行性 | 独立倾向强，情感克制，不轻易信任 | 斥候、独行猎人 |
| **狐系** | 中等，外表精致 | 聪慧狡黠，擅长社交和议价 | 商人、中间人、外交型角色 |

### 2.2 鼠族（Rat-folk）

| 维度 | 内容 |
|---|---|
| **外部特征** | 身高1.0-1.3米，尖鼻、圆耳、细长尾巴。手指异常灵巧 |
| **寿命** | 约40-50年（亚人中最短寿） |
| **繁殖力** | 高。鼠族人口在亚人各亚种中增长最快 |
| **天生能力** | 夜视、狭窄空间穿行、快速反应、精细手工 |
| **社会角色** | 商队经营者、走私者、地下贸易网络的核心掌控者 |
| **组织方式** | 以家族商队为基本单位，无固定聚落。一个家族商队通常20-50人 |
| **与人类关系** | 最擅长与人类做生意的亚人，但也最常被人类欺骗和拖欠货款。以家族为绝对忠诚边界，极少对外人开放 |

> 亚人网络中的"**鼠族商队**"是边境地下经济的命脉。他们掌握着不为人知的走私路线和秘密通道——这也是男主阵营潜在的战略资源。

### 2.3 鳞族（Scale-folk）

| 维度 | 内容 |
|---|---|
| **外部特征** | 体表覆有细鳞（质地接近蜥蜴，非鱼鳞），眼球有瞬膜，手指间有退化蹼痕 |
| **体型** | 与人类相当，偏瘦长 |
| **寿命** | 约80-100年（亚人中最长寿） |
| **天生能力** | 可水下闭气超过30分钟（长期适应后可达1小时以上）、抗寒、对水质变化极其敏感 |
| **社会角色** | 渔民、渡船人、水域守护者 |
| **组织方式** | 以沿河聚落为基本单位，一个聚落通常100-300人 |
| **当前危机** | 🔴 **最受魔潮影响的亚人族群。** 河水污染、鱼群死亡和魔潮洪水导致大量鳞族死亡或被迫放弃传统水域，向南迁徙。他们的古老水域知识对理解魔潮扩散机制至关重要 |
| **与人类关系** | 极少与人类直接接触。生活在沼泽和河流深处，人类大多只知道他们的存在但从未见过 |

### 2.4 有翼族（Winged-folk）——近乎灭绝

| 维度 | 内容 |
|---|---|
| **外部特征** | 背部生有大型羽翼（翼展约3-4米），骨骼中空、体型修长、体重轻。可短距离飞行（单次约数里后需休息） |
| **寿命** | 约50-60年 |
| **历史** | 曾经是天空的掌控者，在高原和山顶建有聚落 |
| **现状** | 🔴 **功能性灭绝。** 数百年来，人类贵族以猎杀有翼族为荣，将其羽翼制成装饰品、羽扇和仪式用具。这种猎杀在贵族文化中被视为高雅娱乐。目前仅可能存在极少数幸存者，隐藏在无人知晓的高海拔废弃巢穴中 |
| **战略价值** | 空中侦察能力在末日中极为珍贵。如果男主能找到并赢得幸存有翼族的信任，将获得巨大的情报和机动性优势——但首先需要克服有翼族对人类的深度恐惧 |

## 三、混血

| 组合 | 后代 | 社会处境 |
|---|---|---|
| **人类 × 亚人（任何亚种）** | 可能生育。后代通常保留部分亚人特征（如较小的兽耳、细鳞斑块或退化翼痕），但天生能力较弱。寿命介于两者之间 | 在人类社会中遭受双重歧视；在纯血亚人社群中也常被视为"不够纯粹"。往往被迫成为边缘人 |
| **不同亚人亚种间** | 生物学上可能但社会性极少发生。各亚种的聚落通常地理隔离 | —— |

## 四、亚人的社会结构

### 基本单位

- 以**聚落**为基本社会单位，无统一国家或王权
- 每个聚落有**长老/祭司**负责传统知识传承和对外决策
- 鼠族以**家族商队**代替固定聚落
- 不存在亚人"国王"或"皇帝"——这是亚人无法组织大规模军事抵抗的根本原因

### 亚人网络

见 `人物角色/07-亚人网络.md`。亚人网络是跨亚种的**松散互助体系**，核心纽带是：
- 共同面对的人类压迫历史
- 贸易与资源交换需求
- 情报共享（特别是关于人类军队和神殿的动向）
- **但不存在军事同盟**——每个聚落优先保护自己的生存

### 信仰

| 信仰 | 信众 | 核心内容 | 与女神信仰的关系 |
|---|---|---|---|
| **源流信仰** | 多数亚人（特别是兽耳族和鳞族） | 世界万物源于同一个生命之流——「源流」。女神和魔王只是源流的两种极端显现，一体两面 | 被神殿定义为**异端**。神殿不积极迫害（需要亚人劳动力），但禁止亚人在人类面前公开讨论源流 |
| **祖灵崇拜** | 部分兽耳族和鳞族 | 逝去的先祖以灵体形式守护聚落。长老兼任祭司，主持祭祖仪式 | 与女神信仰不直接冲突，神殿对此相对容忍 |
| **表面信女神** | 部分与人类贸易密切的亚人 | 为贸易便利而表面接受女神信仰，内心多保留传统信仰 | —— |

## 三、精灵（Elf）——正在消逝的种族

精灵是人类到来之前这片大陆的古老原住民之一。他们不是不朽的——只是活得比人类长得多。

### 基本特征

| 维度 | 内容 |
|---|---|
| **外部特征** | 身高与人类相当（偏纤细），尖耳（长度因氏族而异），肤色苍白至浅小麦色。发色多为银白、浅金、深绿——与他们的森林故乡的颜色呼应。没有"不老的容颜"——精灵会老，只是老得很慢 |
| **寿命** | 约300-400年。极少有精灵活到500年以上——不是因为寿命极限，而是因为他们的世界已经被毁掉，很少有精灵能"自然老死" |
| **天生能力** | 与自然界的深层共鸣——可以感知森林的健康状态、水源的纯净度、动物情绪的波动。这种能力在魔潮污染区域会变成**痛苦的感知**——精灵能"听见"森林在腐烂。在神殿能力鉴定中，这种能力会被归类为高级感知系/概念系 |
| **魔法** | 精灵拥有自己独立的魔法传统——**不同于"女神祝福"体系**。精灵魔法不以"祝福"为中介，而是直接从自然中汲取力量。这使得精灵魔法在神殿眼中是"未经女神许可的力量"——即异端力量 |

### 历史——灭绝战争

精灵不是自然消亡的。他们是**被系统性消灭的**。

| 阶段 | 事件 |
|---|---|
| **共存期**（数百年前） | 人类初至这片大陆时，与精灵有过短暂的共存。精灵教人类识别药草和森林生物——这段记忆几乎已被神殿从历史记录中抹去 |
| **神殿扩张期** | 神殿将精灵的"自然魔法"定义为"未被女神祝福的异端之力"。宣称精灵崇拜"森林中的伪神"——后来这个"伪神"被神殿的叙事改写为"魔王的仆从" |
| **灭绝战争** | 神殿发动"圣伐"——以"清除魔王仆从"为名，对精灵聚落进行系统性毁灭。精灵的森林被焚烧，幸存者被猎杀。精灵的尖耳成为战利品——在贵族收藏中与有翼族的羽翼并列 |
| **残存** | 最后的精灵逃入极北的**苍翠之痕**——一片被魔潮污染包围的残存原始森林。人类无法到达（魔化生物太多），精灵在里面也活不了多久（森林本身正在被魔潮侵蚀） |

### 精灵的现状

| 维度 | 内容 |
|---|---|
| **人口** | 推测不足一万。分布在：苍翠之痕（主体，约七千）、散落各地的流亡者（约两千） |
| **苍翠之痕** | 极北残存森林——精灵最后的家园。正在被魔潮侵蚀——树木从外向内枯死。约十年内，苍翠之痕将不复存在 |
| **流亡者** | 少数精灵在灭绝战争中逃往南方。他们隐藏身份（割去或包裹尖耳），独自或三两成群生活在人类社会的边缘——猎人、药农、流浪艺人 |
| **与亚人的关系** | 精灵与亚人（特别是兽耳族和鳞族）有古老的友好传统——他们在人类到来之前就认识彼此。部分亚人聚落秘密庇护精灵流亡者 |

### 精灵的知识——神殿最恐惧的东西

精灵保存着**人类到来之前的记录**。这意味着：

- 他们知道女神信仰是"后来的东西"——不是自古以来就有的
- 他们可能知道"源流"的原始形态——在神殿将力量二分为"神"与"魔"之前，力量是什么样的
- 他们可能知道魔王的起源——魔王是"一直都存在"的，还是"某件事之后才出现的"
- ⚠️ **他们的存在本身就是神殿历史叙事的证伪**

### 与男主的可能接触途径

| 途径 | 说明 | 可行性 |
|---|---|---|
| **格兰的亚人网络** | 亚人可能知道某个流亡精灵的位置——一个隐藏在某聚落中的精灵药农或猎人 | 🟡 中等 |
| **北方远征** | 如果男主未来需要前往叹息之墙以北——可能在魔潮污染区遇到残存的精灵斥候 | 🟢 低（78天内不太可能） |
| **暮河镇隐姓埋名者** | 边境小镇里可能就有隐藏身份的精灵——割去尖耳的精灵在外表上与纤瘦的人类无异 | 🟡 中等 |
| **被神殿追捕的流亡者** | 类似沈清岚的处境——一个被第四圣殿追捕的精灵可能逃入东南沼泽，与男主阵营相遇 | 🟡 中等 |

### 精灵角色的写作要点

- 不要写成"优雅高贵"的奇幻精灵。他们是**幸存者**——身上有伤疤、眼神警惕、随时准备逃跑
- 精灵的长寿在末日中不是祝福，而是诅咒——他们活过了灭族，现在要活过魔潮
- 精灵的"自然感知"在魔潮中变成了持续的痛苦——他们能感觉到脚下每一寸土地的腐烂
- 对人类的态度：不是仇恨（那太奢侈了），而是**深度警惕 + 偶尔的绝望好奇**——为什么这个种族要毁灭一切？

## 四、矮人（Dwarf）——被囚禁的工匠

矮人是这片大陆上最不幸的种族——不是因为他们弱小，而是因为他们太有价值。

### 基本特征

| 维度 | 内容 |
|---|---|
| **外部特征** | 身高1.2-1.5米，体型敦实，肌肉发达。男女都蓄须（女性胡须较细软）。手部粗壮，手指短而有力——天生的工匠之手 |
| **寿命** | 约150-200年 |
| **天生能力** | 对矿石、金属和地下结构有近乎直觉的感知力。能"听"出岩壁后面的矿脉走向。手臂力量远超同体型人类 |
| **特殊技能** | 锻造、冶炼、矿石鉴定、地下建筑工程——矮人的工艺在这个世界是无与伦比的 |

### 历史——不是因为仇恨而被消灭，而是因为太有价值

矮人的命运与精灵不同。贵族们发现矮人太"有用"了，所以选择了**奴役而非灭绝**。

| 阶段 | 事件 |
|---|---|
| **接触期** | 人类发现矮人居住在地下矿脉丰富的地区。矮人对地表事务不感兴趣，最初与人类和平交易——用锻造品换粮食 |
| **征服期** | 贵族和神殿发现矮人的锻造能力后，发动征服战争。不是要杀光矮人——是要控制矿脉和工匠 |
| **奴役期**（当前） | 矮人被集体迁往西部矿区（铁脊矿区等），在武装看守下进行强制劳动。矮人锻造的武器供应着王国的军队。矮人挖掘的矿石支撑着王国的经济 |

### 矮人的现状

| 维度 | 内容 |
|---|---|
| **人口** | 推测约两万至三万。绝大多数在西部矿区被奴役。少数逃奴散落在东南边境的走私网络或亚人聚落中 |
| **奴役体系** | 以家族为单位被关押。若一个矮人逃跑，其家族全体受罚。这是最有效的控制手段——矮人的家族忠诚度极高 |
| **逃亡者** | 有矮人成功逃脱。他们通常通过矿区的逃亡网络→走私路线→最终到达东南边境或亚人聚落。一个逃亡矮人在自由地带的价值极高——他能锻造出远超人类水平的武器和工具 |
| **与亚人的关系** | 矮人与鼠族商队有地下联系——鼠族走私矿石和矮人锻造品以换取矮人需要的物资。逃亡矮人在亚人聚落中常受到欢迎——他们的锻造技能对生存至关重要 |

### 矮人的价值——在末日中

| 价值 | 说明 |
|---|---|
| **武器锻造** | 矮人锻造的刀剑和箭头在质量上远超人类工匠。在末日中，一把矮人短刀可能是生与死的区别 |
| **建筑与防御** | 灰丘的防御工事如果由矮人设计和建造——会从"脆弱"变为"可防守" |
| **矿石知识** | 矮人知道哪里有矿、如何识别、如何冶炼——这些知识在供应链崩溃的末日中等于独立的生产能力 |
| **地下知识** | 矮人了解地下结构——包括王都的地下排水道网络。他们可能知道**不为人知的王都地下通道** |

### 与男主的可能接触途径

| 途径 | 说明 | 可行性 |
|---|---|---|
| **逃亡矮人铁匠** | 一个逃亡矮人可能在苇水村或暮河镇隐姓埋名做铁匠。男主需要武器时可以找到他 | 🟡 中等 |
| **亚人网络中的矮人** | 格兰或鼠族商队可能认识逃亡矮人工匠。通过以物易物——矮人锻造武器，男主提供粮食或保护 | 🟡 中等 |
| **矿区救援** | 如果需要救援被罚到矿区的同学——可能会同时遇到被奴役的矮人 | 🟢 低（矿区距离太远） |
| **王都地下通道** | 如果男主未来需要返回王都——逃亡矮人可能知道一条不为人知的地下通道 | 🟢 低（当前不需要） |

### 矮人角色的写作要点

- 矮人是**被奴役的工匠**，不是"喜欢喝酒打架的粗犷民族"。他们的幽默感已经被奴役磨尽
- 矮人对家族的责任感是他们行为的核心驱动力——如果能救一个矮人的家族，他会为你锻造到死
- 矮人看待工具和武器的态度是近乎神圣的——每一件锻造品都承载着他们被剥夺的自由
- 矮人女性同样强壮、同样善锻造、同样留胡须——在人类眼中"不辨男女"，这在奴隶市场上反而让她们免受性剥削

## 六、设定原则

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| 把任何一个种族写成统一阵营 | 每个种族内部都有分歧、派系和个体差异 |
| 把所有亚人/精灵/矮人写成受害者 | 个体可以在压迫中做出恶的选择——背叛同族换取自身安全是真实存在的 |
| 把所有人类写成压迫者 | 人类中有对异族友善的个体。善是选择，不是种族属性 |
| 精灵写成"优雅高贵" | 这个世界的精灵是幸存者——伤疤、警惕、破碎。优雅已经被灭绝战争烧掉了 |
| 矮人写成"粗犷搞笑" | 这个世界的矮人是奴隶。幽默感已经随自由一起失去了 |
| 添加更多种族 | 人类、亚人（4亚种）、精灵、矮人——已经足够丰富。不再新增 |
