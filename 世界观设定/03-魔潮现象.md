# 魔潮现象

## 定义

魔潮是魔王力量扩散引发的**大规模自然与生态异变**，表现形式多样，是魔王危机正在恶化的直接证据。

## 已确认现象

| 现象 | 描述 | 首次出现场景 |
|---|---|---|
| **水流逆转** | 河水倒流，违反自然规律 | 男主与米娅在河上时 |
| **暗红光柱** | 天空出现不明来源的暗红色光柱 | 同上 |
| **大洪水** | 伴随魔潮的异常洪水 | 苇水村受灾 |
| **黑云** | 异常天象 | 天空异变 |

## 魔化生物

| 生物 | 特征 | 威胁等级 |
|---|---|---|
| **腐齿鼠** | 感染性强，咬伤米娅 | 中 |
| **腐兽** | 袭击婴儿，能抓伤男主 | 高 |
| **沼鳄** | 沼泽原生生物，可能被魔化 | 高（推测） |
| **水鬼猴** | 沼泽威胁 | 中（推测） |
| **沼犬** | 沼泽威胁 | 中（推测） |

## 魔化特征

- 生物出现**黑鳞**
- 体内或体表形成**魔化晶体**
- **夜间魔化风险增加**（老祭司警告）
- 原本不会出现在某些地区的生物开始迁徙

## 扩散范围

| 区域 | 受影响程度 |
|---|---|
| 北方边境 | 村庄和亚人聚落失联 |
| 东南沼泽 | 腐兽、魔化生物活跃 |
| 苇水村及周边 | 洪水、鱼群死亡 |
| 王都 | 目前主要表现为粮价和社会动荡 |

## 魔化机制

- 亚人老祭司懂得魔化、腐兽、魔潮和古老记录
- 魔化可能与魔王的力量渗透有关
- 具体原理——**待展开**
- 这可能涉及世界观的核心秘密（魔力的来源与本质）
