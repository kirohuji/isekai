# 女神信仰

## 核心信仰

光明女神是这个王国**唯一的官方信仰**。神殿是女神在人间的代言机构。全国范围内不存在被官方承认的其他宗教。

### 女神

| 维度 | 内容 |
|---|---|
| **称谓** | 教典中称「光明女神」或简称「女神」。神殿禁止以凡人之口直呼女神真名（据称真名蕴含力量）。信徒日常使用「女神」「吾主」「光明之母」等敬称 |
| **领域** | 光明、秩序、生命、保护 |
| **对立面** | 魔王——混沌与黑暗的化身 |
| **存在性** | ⚠️ **女神是否真实存在？** 这是故事的核心悬念之一。勇者确实通过"祝福"获得了可验证的力量，但这力量是否来自一位真实存在的人格神——还是某种可以被神殿操控的自然能量——**当前不设明确答案，保持悬念** |

## 创世神话（神殿官方版本）

> 太初有混沌，无形无光。女神自光明中诞生，以意志划分天地，创生万物。
>
> 混沌不甘消退，凝聚残余力量化为魔王——不是生命，而是秩序之敌、存在之否定。
>
> 女神以自身为代价封印魔王，陷入无尽长眠。女神的沉睡之梦维持着世界的秩序。
>
> 当魔王的混沌侵蚀秩序，女神的梦便开始动摇。梦醒之日，若非女神苏醒——便是世界归于混沌。

### 神话的叙事功能

| 功能 | 说明 |
|---|---|
| **将魔王定义为"存在的否定者"** | 魔王不是可以谈判的敌人，而是必须消灭的绝对恶 |
| **将神殿塑造为"维持女神之梦"的唯一合法机构** | 只有神殿掌握与女神沟通（即祝福仪式）的能力 |
| **为召唤仪式提供合法性** | "召唤异世界的灵魂"是"惊扰女神之梦"以求力量——这种"不得已"强化了末日叙事的紧迫感 |
| **暗示代价** | 神话本身就提示"女神以自身为代价"——为神殿隐瞒神力代价提供了文化基础 |

## 教义核心

| 教义 | 经文表述（大意） | 社会控制功能 |
|---|---|---|
| **光明终将战胜黑暗** | "光明所在，暗影无存" | 将一切反秩序行为定义为"魔性" |
| **服从即虔诚** | "凡顺从者，皆在女神之光的庇护之下" | 维护神殿权威，质疑神殿即质疑女神 |
| **力量是女神的恩赐** | "非汝有力，乃女神借汝之臂" | 合法化能力等级制度——你的力量不是你的 |
| **黑暗中的灵魂需要引导** | "迷途者不知暗，故需持光者引之" | 合理化对无能力者和亚人的控制 |
| **牺牲成就救赎** | "为光而逝者，归于女神之梦" | 鼓励征兵、征税、在末日中服从牺牲 |

## 神殿组织体系

### 七大神殿（七大圣殿）

神殿以"**七大圣殿**"为构架，每座圣殿负责不同领域：

| 圣殿 | 所在地 | 核心职能 | 与故事当前关联 |
|---|---|---|---|
| **第一圣殿** | 王都中心 | 最高决策、大神官驻地、总领一切神殿事务 | 整个神殿体系的核心 |
| **第二圣殿** | 王都 | 军事事务、勇者祝福仪式、军队随军祭司 | 周启明等核心勇者与此殿关联 |
| **第三圣殿** | 王都 | 档案管理、历史记录、召唤记录、财务账册 | **沈清岚曾在此抄写**——发现周期性召唤的秘密 |
| **第四圣殿** | 未知（推测北部） | 异端审判、逃犯追捕、异世界检测铃的管理与研发 | 负责追捕沈清岚的机构 |
| **第五圣殿** | 分布于各地 | 农业管理、土地登记、税收、粮食储备 | 战时征收令的执行者 |
| **第六圣殿** | 多地（含暮河镇） | 商业管理、奴隶契约认证、贸易许可发放 | 与商人公会、罗德里克等商人关系密切 |
| **第七圣殿** | 未知（推测边境） | 预言解读、神秘学研究、古代文献保管 | **许安然被分配至此** |

### 内部层级

| 层级 | 称谓 | 人数 | 职能 |
|---|---|---|---|
| **第一层** | **大神官**（Archpriest） | 1人 | 神殿最高权威。身份、姓名、性别均未知 |
| **第二层** | **枢机祭司**（Cardinal Priest） | 7人 | 最高议事会成员，各代表一座圣殿 |
| **第三层** | **祭司**（Priest/Priestess） | 数百人 | 管理地方神殿，执行祝福、预言、审判等职能 |
| **第四层** | **修士 / 修女**（Monk / Nun） | 数千人 | 基层神职人员，抄写、护理、勤务、基层管理 |
| **第五层** | **见习修士**（Novice） | 不定 | 新人。部分是被分配来的召唤者 |

## 核心仪式与道具

### 召唤仪式

见 `世界观设定/01-召唤事件.md`。大型仪式需神殿与王室联合举行。消耗巨大——具体代价待展开（可能涉及生命力或其他不可再生资源）。

### 祝福仪式

- 赋予召唤者以神力的核心仪式
- 初次祝福量 = 能力鉴定的天生容量 + 女神（或仪式本身）根据某种规则注入的力量
- 追加祝福量 = 服从度与贡献度的评估结果
- ⚠️ 可能存在**反向仪式**——撤回或减少特定对象的祝福

### 异世界检测铃

| 维度 | 内容 |
|---|---|
| **功能** | 对异世界人产生反应（发出铃声或光晕），用于追踪逃逸的召唤者 |
| **原理** | 检测召唤过程中留在异世界人身上的"世界间隙痕迹"——穿越世界边界时残留在灵魂上的能量印记 |
| **局限性** | 已登记召唤者（如男主）身上的痕迹已被神殿备案——可用备案者的信息在人口普查和身份核验中为未备案者提供一定掩护 |
| **有效范围** | 约数十米半径（具体范围取决于检测铃的品质和仪式强度） |
| **制造者** | 第四圣殿的专门工匠——制作方法属于神殿机密 |

### 预言仪式

- 第七圣殿负责
- 预言的发出者需要特殊天赋（如许安然）
- ⚠️ 预言的代价极高——许安然的失明可能与预言行为直接相关：「看到了不该看到的东西，所以被收回了光明」

## 女神信仰的控制机制

| 机制 | 运作方式 | 效果 |
|---|---|---|
| **力量垄断** | 只有神殿能主持祝福仪式。离开神殿=失去力量补充来源 | 勇者被绑在神殿体系上 |
| **身份控制** | 神殿管理所有召唤者登记和奴隶契约，控制"谁是合法存在"的定义权 | 神殿可以合法化或非法化任何人 |
| **信息垄断** | 神殿是唯一有权解读预言和古代文献的机构 | 真相被过滤后才到达民众 |
| **恐惧管理** | "魔王十年后全面战争" → 增税征兵；"红月预言" → 引发恐惧 → 人民更依赖神殿 | 恐惧 = 神殿权力的燃料 |
| **司法权力** | 神殿拥有独立的审判权，可绕过世俗法庭审判"异端"和"逃犯" | 法律体系无法制约神殿 |

## 与王室的关系

| 维度 | 说明 |
|---|---|
| **理论关系** | 王权神授——国王的统治权来自女神，登基需神殿认证。神殿在理论上高于国王 |
| **实际关系** | 合作与制衡并存。大部分时间神殿占优。战时神殿权力进一步扩大。 |
| **当前张力** | 如果王室发现神殿隐瞒魔王真相（特别是"十年后"的时间谎言），合作关系可能破裂。国王可能正在暗中收集神殿的情报 |

## 亚人的信仰冲突

亚人的"**源流**"信仰与女神一神论存在根本性对立：

| 维度 | 女神信仰 | 源流信仰 |
|---|---|---|
| 世界观 | 一元善（女神）vs 一元恶（魔王） | 一体两面——女神与魔王是同一力量的不同表现 |
| 力量观 | 力量是神的恩赐，人无权自行获取 | 力量属于所有生命，只是不同生命接通源流的能力不同 |
| 代表人物 | 大神官 | 亚人老祭司 |

> 亚人老祭司的知识可能包含神殿试图掩盖的真相：**力量的原始统一性**——即神力与魔力同源。

神殿将源流信仰定义为"异端"，但出于对亚人劳动力的需要，不进行系统性迫害——除非有政治需要（如罗德里克举报聚落"私藏逃奴"，就给了士兵抓人的"合法"理由）。

## 设定原则

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| 把神殿写成铁板一块的纯粹邪恶组织 | 基层神职人员可能真心信仰女神、真心想帮助他人。恶来自于制度性的权力集中和高层选择 |
| 在故事早期揭示女神真相 | 女神的存在性应保持模糊——让读者与男主一样无法确定 |
| 把女神设定为明确真实且完全善良的 | 即使女神真实存在，她的意图也可能并非人类所理解的那样 |
| 神殿全知全能 | 神殿也有盲区——男主逃过搜查就是证明。神殿会犯错、会被欺骗、内部也有矛盾 |
