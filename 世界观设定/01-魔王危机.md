# 魔王危机

## 公开说法

神殿宣称：魔王将在**十年后**发动全面战争。

王国以此为理由：
- 增加税收
- 征粮
- 征兵
- 扩建勇者宫殿
- 强化神殿权力

## 真实迹象

魔王危机**不是完全虚构**。

### 已发生的异变

| 类型 | 具体现象 |
|---|---|
| **社会经济** | 王都粮价上涨、多次暴乱 |
| **自然灾害** | 地震、暴雨、干旱、河流倒灌 |
| **边境危机** | 北方村庄和亚人聚落失联 |
| **生态异变** | 魔物向南迁徙、腐兽出现在不该出现的地区、植物反季节枯萎、牲畜发狂 |
| **水质异变** | 河鱼死亡并出现黑鳞与魔化晶体 |
| **天空异象** | 暗红光柱、黑云 |
| **水系异变** | 魔潮导致水流逆转、大洪水 |

## 亚人祭司的判断

> 亚人老祭司认为**战争已经开始**，只是王都尚未承认。

这意味着：
- 异变不是「魔王苏醒的征兆」，而是「魔王已经在行动」
- 王都和神殿要么不知道真相，要么故意隐瞒
- 亚人因为生活在边境和沼泽，比人类更早感受到变化

## 双重真相

> **神殿在利用魔王危机，但魔王危机本身又确实存在。**

这是整个故事最基本的叙事原则之一。不能简单地把神殿写成纯粹编造危机的骗子，也不能忽略神殿确实在撒谎的事实。
