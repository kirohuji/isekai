# 世界概述

## 基本信息

- **世界名称**：大陆称为「**埃瑟兰**」（Aesland），但普通民众通常只说"大陆"或"王国境内"
- **当前舞台**：圣光王国及其东南边境。王国首都简称「**王都**」
- **时代背景**：末日降临前夕的封建神权社会
- **当前年份**：**光明历 847 年**（圣光王国以神殿建立之年为纪元元年）
- **当前季节**：夏末至初秋（沼泽湿热，北方已有寒意）

## 天文与历法

### 双月体系

有两座月亮：

| 月亮 | 外观 | 运行周期 | 说明 |
|---|---|---|---|
| **银月** | 银白色，体积较大 | 30天 | 历法基础，一个月即一次银月盈亏 |
| **红月** | 正常情况下呈暗铜色，体积较小 | 39天 | 与银月不同步运行 |

### 红月重合现象

双月重合约每 **78天** 发生一次（30与39的最小公倍数）。重合时红月在夜空中短暂变为红色。

但关键的是：**每一次重合，红色都比上一次更深、更持久。**

这不是突然事件——是**渐进式的天文异变**。神殿的记录显示：

| 过去的几次重合 | 红色深度 | 持续时间 |
|---|---|---|
| 约一年前 | 微弱的淡红 | 约一炷香 |
| 约半年前（召唤事件前后） | 明显的暗红 | 约半个时辰 |
| 上一次 | 刺目的血红 | 近一个时辰 |
| **下一次（？）** | **未知** | **未知** |

### 许安然的预言与倒计时

许安然在第七圣殿的预言记录中反复出现同一句话：

> **「当红月彻底染血，王都将在那一夜化为废墟。」**

根据她的记录和神殿的推算——按当前的红色加深速度——红月将在**大约两年后**（约720天）达到"彻底染血"的临界点。

> ⚠️ 这不是精确的720天倒计时——而是**含不确定性的估算**。也许更早。也许稍晚。但"两年"是当前最合理的推算。每一天红月都可能比预期更红一点，每一次重合都是对神经的折磨。

### 历法

| 维度 | 内容 |
|---|---|
| **一年** | 360天，12个月，每月30天 |
| **季节** | 春、夏、秋、冬各约90天 |
| **纪元** | **光明历**——以神殿在圣光王国建立第一圣殿之年为元年。当前：光明历 847 年 |
| **月份命名** | 以数字称呼——"一月""二月"……"十二月"。神殿历法也有宗教性月名（如"降临之月""祝福之月"等），但民间只用数字 |

## 种族寿命一览

| 种族 | 平均寿命 | 成年年龄 | 备注 |
|---|---|---|---|
| **人类**（本地） | 60-80年 | 16-18岁 | 贵族寿命较长（营养和医疗），底层较短 |
| **人类**（召唤者） | 同地球人类 | 18岁 | 未观察到寿命因穿越而改变 |
| **兽耳族**（亚人） | 60-70年 | 14-16岁 | 狼系偏长，猫系偏短 |
| **鼠族**（亚人） | 40-50年 | 10-12岁 | 亚人中最短寿，但繁殖力最高 |
| **鳞族**（亚人） | 80-100年 | 18-20岁 | 亚人中最长寿 |
| **有翼族**（亚人） | 50-60年 | 14-16岁 | 近乎灭绝 |
| **精灵** | 300-400年 | 25-30岁 | 长生但正在被灭绝。活到老死的精灵极少 |
| **矮人** | 150-200年 | 20-25岁 | 被奴役者寿命显著缩短 |
| **上位魔族** | 500年以上（部分永生） | 因类型而异 | 魔王、魔将、魅魔等。不被杀死则几乎不自然死亡 |
| **下位魔族** | 50-100年 | 因类型而异 | 消耗型战力，寿命短，繁殖（或生成）快 |
| **魔化生物** | 无固定寿命 | — | 不是自然生物。通常在魔化后数月到数年内完全异形化并死亡 |

## 作品类型

黑暗奇幻 / 异世界生存 / 政治阴谋 / 冒险 / 恋爱 

## 核心主题

| 主题 | 说明 |
|---|---|
| 无力量者的生存之道 | 没有力量的人如何利用信息、商业、制度和可信赖的人活下去 |
| 权力与人性的放大 | 权力不会自动创造恶人，但会放大人性，并奖励顺从、沉默和作恶 |
| 双重真相 | 神殿可能在利用魔王危机，但魔王危机本身又确实存在 |
| 信任的成本 | 信任不是善良，而是一种必须承担失败风险的长期投资 |
| 保护与自主 | 保护一个人，不等于替对方决定一切 |
| 末日经济 | 最值钱的资源不是金银，而是土地、水、粮食、药品、交通、情报和可靠的人 |
| 非天选英雄 | 男主不是天选英雄，而是依靠判断、责任感和组织能力逐渐建立自己的领地 |
| 领地崛起 | 从一块荒废高地和四个人的微型据点开始，在末日中建立一方势力 |
| 国家间博弈 | 人类的敌人不仅是魔王——还有彼此 |

## 故事基调

| 维度 | 原则 |
|---|---|
| **严肃性** | 没有无代价的成功，没有突然降临的万能力量，没有纯粹为推动剧情而降智的敌人 |
| **角色智商** | 角色会撒谎，会隐瞒，会因利益暂时合作。好感≠忠诚，善意≠可靠 |
| **恋爱线** | 必须慢热。 |
| **黑暗面** | 奴隶制度、人口交易、神殿对女性召唤者的控制等属于黑暗社会背景，可进行露骨性暴力描写，但须有叙事意义 |
| **国家政治** | 王国之间不是简单的"盟友或敌人"。利益关系错综复杂。内斗在末日中继续 |

## 末日时代最有价值的资源

> **土地 > 水 > 粮食 > 药品 > 交通 > 情报 > 可靠的人**

金银不再是最重要的通货。
