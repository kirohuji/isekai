## 角色原型库（供参考）

以下原型并非穷举，也不应机械套用。每个角色的具体行为应由其穿越前人格+穿越后处境+力量变化共同决定。

| 原型 | 描述 | 示例 |
|---|---|---|
| **理想主义者** | 真心相信神殿和勇者使命，想拯救世界 | |
| **幸存者** | 为活下去什么都可以做，但没有主动作恶 | |
| **投机者** | 看准强者依附，积极表现以换取更好待遇 | 孙茂 |
| **权力成瘾者** | 原来被欺负的，获得力量后反过来压迫 | 可能类型 |
| **沉默纵容者** | 知道不对，但不敢说，用沉默保护自己 | |
| **暗中反抗者** | 表面服从，私下帮助弱者或传递情报 | 值得救援 |
| **背叛者** | 原与男主关系不错，但在压力下出卖 | 潜在戏剧冲突 |
| **牺牲者** | 已死亡或失踪 | |
| **被彻底摧毁者** | 精神和身体都被摧毁，已无自主意识 | |
| **真正的信徒** | 被神殿洗脑，真心相信所做的一切 | |
| **建立自己势力者** | 不甘依附任何人，试图建立第三方阵营 | |

## 写作提醒

1. 新同学不需要一次性全部创建。根据剧情需要逐步引入。
2. 每引入一名新同学，都给当前故事增加一个**道德抉择**或**资源取舍**。
3. 「可以救援」不等于「必须救援」。男主需要评估每次救援的成本和风险。
4. 被救援的同学不一定会感激或忠诚。有些人可能怨恨男主没有更早救援。
5. 沈清岚的名单信息是分段的——她对某些人的了解更多，对某些人只知道位置。
