# 艾米莉亚

## 基本信息

| 维度 | 内容 |
|---|---|
| **姓名** | 艾米莉亚（Sophie） |
| **性别** | 女 |
| **年龄** | 约18-20岁 |
| **身份** | 异世界本地人，灰绳市场女奴，现为旧鹿角旅馆记账员 |
| **外貌** | 短发，发尾不整齐（自己用剪刀修的）。重伤恢复中，左眼曾肿胀。长相与沈清岚有几分相似——这是男主冲动举牌的原因 |

## 背景

- 父亲曾在城南开布铺，家境尚可
- 约十岁时铺子倒闭、父亲去世，被债主拉去抵债
- 此后辗转多个主人：做过厨房帮工、洗衣、带孩子
- 最后一位主人因赌博将她卖给人贩子，流入灰绳市场
- **从未受过正式教育**，不会写字，但会看数字

## 核心能力

| 能力 | 说明 |
|---|---|
| 算数和记账 | 数字感好，能快速理清收支、分类账目 |
| 观察力 | 在多次转手中学会了判断人的意图 |
| 耐受力 | 能忍耐疼痛、饥饿和恶劣条件 |
| 谈判 | 知道自己的劳动力值多少钱，敢于问清条件 |
| 识字（学习中） | 男主正在教她。从数字开始，逐步扩展到货名和日期 |

## 性格

- **自卑**：内心深处觉得自己不配被善待。从十岁被拉去抵债起，她学到的最牢固的一课就是——自己不值钱。被男主买下后，她始终有种"随时会被嫌弃"的不安，习惯性地压低自己的存在感
- **不想麻烦任何人**：能自己做的事绝不开口。痛不说，饿不说，冷不说。被照顾时会不安，第一反应是"我是不是添麻烦了"
- **对男主百依百顺**：从不反驳，从不拒绝，从不提要求。他说什么就是什么。不是因为奴隶契约——是因为她怕。怕自己多说一句、多走一步，眼前这好不容易抓住的安稳就会碎掉
- **外平内敏**：说话的语调很平，不好听出情绪——这是多年被转手练出来的生存本能。但内心敏感，男主的每一个表情、每一次沉默、每一句没说完的话，她都会反复琢磨
- **生理性喜欢男主，但拼命掩饰**：她自己都不太明白那是什么。只是心跳会变快，耳朵会发烫，会在男主转身后盯着他的背影出神。但她觉得这种感情太荒唐了——一个被从奴隶市场捡回来的人，有什么资格。于是她藏得很深，只在没人看的时候才允许自己的目光多停几秒
- 对"被当人看"这件事既渴望又警惕，但警惕正在被男主一点一点融化
- 在被叫"喂"的漫长日子后，第一次有人问她的名字，也第一次有人给了她一个名字

## 与男主的关系

- 男主因她长相与暗恋对象沈清岚相似而冲动买下
- 男主提出的条件：正常雇工，包吃住，工资与8银币对冲
- 她不会写字，男主承诺教她
- **关系本质**：她从"救命恩人"的角度看待男主，感激中掺杂着连自己都不敢承认的生理性喜欢。她会不自觉地靠近他，又在意识到之后立刻退开半步
- **她的隐藏感情**：
  - 心跳会因为他在场而变快，耳朵会红
  - 会偷偷看他——在算账的时候、在吃饭的时候、在他靠在门框上的时候
  - 被铜镜里发现自己在看他时，会迅速移开眼，假装在看别的东西
  - 她不知道这叫什么。或者她知道，但不敢叫出来
  - 她觉得自己没资格。一个被卖了好几次的奴隶，一个连字都不会写的女人，喜欢自己的买主——这话说出去连她自己都觉得可笑
- **她的顺从**：不是因为奴隶契约要求她顺从。是因为她怕失去。怕自己不够乖、不够有用、不够安静，就会被嫌麻烦，就会被送走。她已经被送走太多次了
- **她的自卑与男主的善意之间的张力**：男主每对她好一分，她心里就多一分不安——"他为什么对我这么好？""我值得吗？""他是不是对谁都这样？""是不是因为我长得像那个人——"

## 说话风格

- 声音轻，比一般人说话低半格，像是怕吵到谁
- 句尾经常往下掉，像是在说出口之前就自己先泄了气
- 陈述事实多过表达感受——但现在的"陈述事实"里多了一层"确认自己没做错"的小心
- 对男主说话时经常加"对不起""谢谢""麻烦你了"，频率高到玛莎有一次忍不住说"你能不能别老道歉"
- 偶尔露出被磨钝了的无所谓——在说到自己的过去时
- 被男主夸了会低头，耳朵尖红，然后迅速转移话题

## 在故事中的作用

- 帮男主分担记账工作，把莱恩的乱账理出头绪
- 以异世界本地人视角提供对奴隶制度、市场规则的理解
- 与沈清岚形成对照：一个是从未受过教育的本地奴隶，一个是穿越而来被困神殿的同学。两人长相相似，但命运截然不同
