# 女性角色与感情线规划

## 总览

本作标签含「**多女主潜在线**」。以下列出所有可能发展感情线的女性角色——包括**正史线**中可能出现的，以及**IF线**专属的。

> ⚠️ 原则：所有感情线必须**慢热**。好感≠恋爱。告白≠关系确立。每个女性角色都应有独立于男主的完整人格、动机和成长线。

## 感情线的写作原则（多女主）

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| 后宫倒贴——女性角色无条件喜欢男主 | 每个女性角色对男主的好感应有**独立的、完整的心理逻辑** |
| 女性角色之间无互动或只有争风吃醋 | 女性角色之间应有独立的关系——沈清岚和米娅的姐妹感、蕾娜和沈清岚可能互相尊重但不亲近、苏瑶可能崇拜沈清岚 |
| 告白即确立关系 | 每一对感情线都是独立的慢热线。男主同时面对多份感情时的犹豫和愧疚本身就是叙事张力 |
| 把多女主写成"收集品" | 每个女性角色都是独立的人，有自己的目标和选择。她们可以选择离开 |
| 感情线遮蔽主线 | 末日倒计时是永远的背景压力。感情必须在末日背景下发展——而不是末日在感情戏的间隙出现 |

添加魅魔感情线,魅魔的是那种需要每天和男主做爱,回复精力,男主不能离开她,双方绑定,男主成为魔王,然后打败魔王成为魔王,反攻大陆,收复其他女人作为附属品
