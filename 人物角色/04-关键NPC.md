# 关键NPC

## 旧鹿角旅馆

### 玛莎
| 维度 | 内容 |
|---|---|
| 身份 | 厨娘，寡妇 |
| 性格 | 嘴硬心软。切菜速度暴露情绪。不骂人=真正在担心 |
| 关键行为 | 多煮一碗麦糊给艾米莉亚。把旧外套"太短了"给艾米莉亚。通过费恩获取伤药。围巾解给艾米莉亚。男主走后每天照做四碗麦糊 |
| 现状 | 留守旅馆管账 |

### 莱恩
| 维度 | 内容 |
|---|---|
| 身份 | 清洁工兼收账 |
| 性格 | 胆小但诚实。会犹豫、贪小便宜但本质不坏 |
| 关键行为 | 把积蓄藏在柜台裂缝里。交账时铜币擦得比以前干净。给男主一枚"还没发生的运费"铜币。曾在灰手外围跑腿 |
| 现状 | 留守旅馆。搬到储物间睡 |

### 哈罗德
| 维度 | 内容 |
|---|---|
| 身份 | 独眼退伍老兵。旅馆夜间守卫 |
| 背景 | 曾在王国军队服役，在北方边境失右眼。无抚恤退伍→佣兵散活→旅馆守夜→重返矿区护卫 |
| 性格 | 沉默。磨刀=处理情绪。用行动代替言语 |
| 关键行为 | 给艾米莉亚带棉衣（"商队不要的"——他自己穿正好）、梳子（"商队发的"——北门商队不发梳子）、坐在走廊尽头守她的门。男主出发前留了五银币 |
| 现状 | 前往矿区做商队护卫。两个月后回 |

## 灰手盗贼行会

灰手没有正式头目——是松散犯罪网络。核心人物不固定。莱恩曾在灰手外围跑腿。灰手小偷和乞丐是街上最不显眼的信息收集者。
