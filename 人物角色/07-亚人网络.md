# 亚人网络

## 组成

- 多个亚人聚落（具体位置未公开）
- 鼠族商队
- 亚人猎人
- 药农
- 走私路线与秘密通道

## 立场

- 对人类王国**深度不信任**
- 历史上多次被人类商人欺骗、拖欠货款、举报和抓捕
- 会基于**合同、人情和共同敌人**进行合作
- 拥有自己的贸易网络和情报系统

## 与人类的冲突历史（以格兰聚落为例）

| 事件 | 说明 |
|---|---|
| 贸易 | 亚人聚落向罗德里克出售银叶草 |
| 欺骗 | 罗德里克拖欠货款 |
| 背叛 | 罗德里克举报聚落私藏逃奴 |
| 镇压 | 士兵抓走七人，包括格兰的妻子 |

## 格兰的纽带作用

- 格兰是当前男主与亚人网络的关键纽带
- 他并非代表整个亚人网络，而是其中一支
- 格兰优先保护家人（妻子和米娅），合作可能因家庭原因中止
- 亚人老祭司懂得魔化、腐兽、魔潮和古老记录

## 战略价值

| 维度 | 说明 |
|---|---|
| 路线 | 走私路线和秘密通道，不受王国控制 |
| 情报 | 亚人对魔潮和异变的了解比人类更早更多 |
| 武力 | 猎人、弓箭手等非正规战斗力 |
| 贸易 | 药草（如银叶草）和其他边境物资 |
| 身份 | 可以为逃亡者提供不在人类登记系统中的身份 |
