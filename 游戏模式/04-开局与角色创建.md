# 开局创建与角色初始化

## 开局方式

游戏支持两种开局方式：

### 自由开局（沙盒模式）
需要开局的介绍,然后从
从"召唤后拿到120银币"开始。**所有故事走向由你的选择决定。**

**初始状态**：
- 位置：王都·召唤广场
- 时间：光明历847年 三月 第1天 · 上午
- 队伍：男主（仅1人）
- 红月倒计时：约900天（比正史线多约200天缓冲）

**初始资源**：
| 资源 | 数值 |
|---|---|
| 银币 | 120 |
| 粮食 | 0天（未购买——王都可以随时买到） |
| 药品 | 0 |
| 生命 | 100/100 |
| 体力 | 100/100 |
| 精神 | 90/100 |

**初始技能**：
| 技能 | 等级 |
|---|---|
| 语言·王国通用语 | 0（未学——需要花钱学） |
| 经商·账目计算 | 2 |
| 策略·危机判断 | 2 |
| 社交·谈判 | 1 |
| 知识·现代医学基础 | 2 |
| 生存·野外识别 | 0 |
| 战斗·基础防身 | 0 |

**初始标记**：无。所有标记通过游戏过程自然产生。

> **自由开局就是IF线沙盒。** 你选择走哪条路：
> - 买旅馆 → 走正史方向
> - 加入佣兵团 → IF-C线
> - 买亚人奴隶 → IF-B线
> - 去北方 → IF-E线
> - 配合神殿 → IF-A线
> - 做商人 → IF-D线
> - 或者混合以上——自由沙盒允许你做任何合理的选择

## 难度设置

| 难度 | 初始银币 | 粮食 | 药品 | 事件概率 | 战斗难度 |
|---|---|---|---|---|---|
| **故事模式** | 120 | 7天 | 3 | 负面事件-20% | 逃跑成功率+30% |
| **生存模式**（推荐） | 120 | 3天 | 0 | 标准 | 标准 |
| **末日模式** | 80 | 0天 | 0 | 负面事件+20% | 逃跑成功率-20% |

## 可选的男主属性加点

以下属性影响**技能检定**和**特定行动的成功率**。自由开局可分配**6点**。

| 属性 | 说明 | 影响 |
|---|---|---|
| **洞察** | 观察力、察言观色 | 探索发现、识破谎言 |
| **冷静** | 压力下的判断力 | 危机决策、谈判 |
| **坚韧** | 身体和心理承受力 | 受伤恢复、抵抗精神消耗 |
| **魅力** | 社交说服力 | 谈判、NPC初始好感 |
| **智谋** | 策略和计算能力 | 商业、计划制定、制度漏洞利用 |

> 正史开局默认加点（已锁定）：洞察2、冷静2、坚韧1、魅力0、智谋1。

### 自由开局提示词

```
光。

你睁开眼睛的第一秒，看到的是一片刺眼的白。

不是学校的天花板。不是寝室的窗帘。是石头。是穹顶上密密麻麻的符文，亮得像烧红的铁丝。空气里有烧灼金属的味道。你的手掌按在冰凉的石板上。石板上有一圈一圈的花纹——不是花纹，是刻痕。几百道刻痕，从你脚下一直延伸到圆形大厅的边缘。

周围全是人。你认识他们。高三十班的三十个男生的脸、二十个女生的脸——穿着穿越前最后一节课的衣服。有人在哭。有人在喊。有人还闭着眼，大概以为这是一场梦。

大厅出口站着一排穿灰袍的人。他们的手里没有武器，但他们的袍子下面有铃铛。铃没有响——只是在发烫。

其中一个人走到你面前。他的眼睛从你身上扫过去。你感觉不到他在看你——你在被他衡量。

"无能力。"他说。声音没有任何情绪。"下一个。"

一块木牌被塞进你手里。上面写着你的名字——中文。旁边有人用墨水补了一行你不认识的文字。后来你才知道，那行字的意思是——

"安置金：120银币。"

半年后，你会站在这里回想这一刻。你以为那是结束。其实那是开始。

但现在，你还不知道。

你现在只知道你手里有120枚银币，脚下是一座陌生的城市，周围是一种你听不懂的语言。太阳压在你头顶上。空气是陌生的。你饿了。

你要做什么？
```
