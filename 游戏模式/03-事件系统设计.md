# 事件系统设计

## 核心思路

**不是手写1000+个事件。而是设计40-50个事件模板 + 参数化条件，通过不同的参数组合（时间、地点、人物、关系值、标记状态）动态生成1000+种独特事件。**

> 模板 = 叙事的骨架。参数 = 叙事的血肉。

---

## 事件模板结构

每个事件模板包含：

```json
{
  "id": 101,
  "name": "街头遭遇·可疑人物",
  "category": "exploration",
  "base_text": "你在{location}的街上走着。{time_desc}的{weather}中，你注意到{target_desc}。",
  "condition": {
    "location_type": ["city", "town"],
    "time_block": ["上午", "下午", "傍晚"],
    "min_mental": 30,
    "random_chance": 25,
    "not_flag": "已触发过_街头遭遇_本轮"
  },
  "choices": [
    {
      "text": "保持距离，观察对方",
      "skill_check": {"skill": "策略·危机判断", "dc": 10},
      "success": {"info": "你注意到对方的{clue}。", "flag_set": "发现可疑人物线索"},
      "failure": {"text": "对方似乎察觉了你的目光，匆匆离开了。"}
    },
    {
      "text": "上前搭话",
      "min_mental": 50,
      "outcome": "trigger_event_template {random_npc_encounter}"
    },
    {
      "text": "绕道而行",
      "outcome": "无事发生。你安全地离开了。"
    }
  ]
}
```

## 事件分类与模板数量

| 大类 | 子类 | 模板数 | 参数变体数 | 总事件数（估算） |
|---|---|---|---|---|
| **探索** | 街头遭遇、野外发现、废弃建筑、隐秘地点 | 8 | 5-10 | ~200+ |
| **社交** | NPC对话、请求帮助、秘密交换、冲突 | 10 | 5-15 | ~300+ |
| **资源** | 价格波动、物资发现、短缺危机、交易机会 | 6 | 5-8 | ~100+ |
| **生存** | 饥饿、疾病、受伤、天气灾害、魔潮 | 5 | 5-10 | ~100+ |
| **战斗/逃生** | 魔物遭遇、追捕者、强盗、腐兽 | 5 | 8-15 | ~150+ |
| **感情** | 好感事件、冲突事件、告白后续、吃醋 | 6 | 3-5 | ~80+ |
| **剧情** | 主线推进、关键NPC行动、重大选择 | 8 | 2-4 | ~50+ |
| **日常** | 琐事、随机小事件、天气变化 | 4 | 5-10 | ~80+ |
| **计数** | | **52模板** | | **约1060事件** |

## 事件模板详细设计（核心20个示例）

### 探索类

**T-001 路边发现**
```
基础文本：你在{location}附近走着，脚下的泥地被什么东西硌了一下。
触发条件：location_type=outdoor, random_chance=20%
选择：
  1. 挖开看看 → 可能发现：少量铜币/生锈的工具/一块奇怪的石头（后续事件线索）
  2. 不管它 → 无视
```

**T-002 废弃建筑的线索**
```
基础文本：你路过一栋{abandoned_building}。门板已经掉了，里面黑洞洞的。
触发条件：location_type=border, random_chance=15%
选择：
  1. 进去探索 → 发现：{旧日记本/破损的武器/藏匿的物资/躲藏的{race}逃奴}
  2. 继续赶路 → 无视
  3. 在门口做标记，改天带人来 → 记住位置
```

**T-003 异常痕迹**
```
基础文本：地面上的{track_type}引起了你的注意。{track_detail}
触发条件：location_type=wild, 策略技能>=2
参数：
  - track_type：脚印/血迹/拖拽痕迹/马车辙印
  - track_detail：很新鲜/被雨水冲过/往{方向}去了/旁边有掉落的{物品}
选择：
  1. 顺着痕迹追查 → 可能触发：发现受伤的旅行者/找到失窃物资/撞上魔物
  2. 记录位置后离开 → 情报记录
```

### 社交类

**T-004 NPC主动搭话**
```
基础文本：{npc_name}走到你面前。{npc_action}
触发条件：npc在附近, affection>=-20, random_chance=30%
参数（根据npc性格）：
  - 沈清岚："她的手指无意识地卷着袖口的线头。'我想跟你谈谈{topic}。'"
  - 格兰："他把弓靠在墙上。'有个事。'{topic}。"
  - 米娅："她的耳朵竖了起来。'喂——'"
选择：
  1. 认真听 → 推进关系，获取情报
  2. 改天再说 → 好感-2
```

**T-005 求助者**
```
基础文本：一个{requester_desc}突然跪在你面前。'{plea_text}'
触发条件：location_type=town|village, reputation>=10
选择：
  1. 帮助 → 消耗{资源}，获得{感激/情报/未来的回报}
  2. 拒绝 → 声望-5。可能影响后续事件
  3. 问清楚要帮什么 → 获取信息后再决定
```

**T-006 紧张对峙**
```
基础文本：{npc_name}和{npc2_name}之间的气氛突然变得很紧张。{conflict_reason}
触发条件：两个NPC在同一地点，某方好感<对方
参数：冲突原因——资源分配/过去的恩怨/对男主的看法分歧
选择：
  1. 调停 → 社交技能检定
  2. 站在{npc_name}一边 → 好感+10/-10
  3. 让他们自己解决 → 好感各-3
```

### 资源类

**T-007 物价波动**
```
基础文本：你注意到{market}的{goods}价格比昨天{price_change}了。
触发条件：location=market, 每3-7天触发一次
参数：
  - goods：粮食/药品/盐/木材/武器
  - price_change：涨了两成/跌了一半/根本没货了
影响：购买和出售价格调整
```

**T-008 意外收获**
```
基础文本：格兰从外面回来，肩上多扛了一个麻袋。'{source}那边搞到的。{item}。'
触发条件：格兰在队伍中, random_chance=10%(每3天)
参数：source=鼠族商队/旧聚落/猎人网络, item=药品/银叶草/箭头/情报
```

### 生存类

**T-009 伤口恶化**
```
基础文本：{受伤部位}的伤口不对劲。边缘发红，按下去有黄色液体渗出来。这不是在愈合——这是在发炎。
触发条件：health<70, 无药品, random_chance=30%
选择：
  1. 用清水冲洗 → 可能无效，可能缓解
  2. 找药草 → 消耗时段，需野外识别技能
  3. 忍一忍 → health-5, mental-10
```

**T-010 粮食见底**
```
基础文本：你打开粮袋。底下的碎屑已经多过能吃的了。{food_days}天。这是你们所有人还活着的天数。
触发条件：food_days<=3, 每2天触发一次变体
选择：
  1. 立刻出去找粮食 → 优先供应采集/交易/狩猎
  2. 减少每人配给 → food_days延长，但全员体力-10/天
  3. 向苇水村求救 → 声望-15（消耗人情）
```

**T-011 魔潮逼近**
```
基础文本：{time_of_day}的天空在{compass_direction}方向出现了一道暗红色的光。和上次一样——但这次看起来更近了。
触发条件：red_moon_countdown % 78 < 5, location_type=border
影响：mental-15。当晚魔化生物出现概率翻倍。
```

### 战斗/逃生类

**T-012 腐兽遭遇**
```
基础文本：树丛后面传来一声低沉的嘶吼。不是人的声音。两对暗黄色的眼睛从灌木缝隙里盯着你。腐兽——两只。体型不大，但嘴边的泥浆里混着黑色的东西。
触发条件：location_type=wild, time_block=傍晚|夜晚
选择：
  1. 慢慢后退 → 策略检定。成功则安全撤离。失败则进入战斗
  2. 站着不动 → 腐兽可能离开（概率30%），也可能扑上来
  3. （格兰在队伍中时）让格兰射箭 → 格兰消耗1支箭。一只腐兽被射中。另一只扑上来
```

### 感情类

**T-013 沉默的靠近**
```
基础文本：{love_interest}在你身边坐了下来。什么都没说。只是坐着。距离近到你能{感知细节}。
触发条件：affection>40, location_type=safe, random_chance=20%
参数（不同女主不同细节）：
  - 沈清岚："闻到她袖口上的草药味——那是她帮你换绷带时沾上的"
  - 蕾娜："她的匕首搁在膝盖上。她从不把匕首放在够不到的地方——但这次它离你比离她近"
选择：
  1. 也什么都不说 → 感情+5（无声的默契）
  2. 开口说话 → 触发特定对话
```

**T-014 旧伤被碰到**
```
基础文本：你不小心碰到了{love_interest}的{旧伤部位}。她的身体绷了一下——只有半秒——然后恢复了正常。
触发条件：affection>30, random_chance=15%
选择：
  1. 道歉 → 她摇头。"已经不怎么疼了。"——但你知道她在说谎
  2. 假装没注意到 → 她松了口气——但眼里有一闪而过的失落
  3. 轻轻握住她的手 → 感情+15（只有一次——重复无效）
```

### 剧情类（主线推进）

**T-015 周启明的动向**
```
基础文本：一个从暮河镇方向过来的人带来了消息——'{news}'
触发条件：已标记"周启明封锁猎人路"，每5-10天触发一次
参数（news）：
  - "那个光系勇者又调了三个守卫过来。"
  - "听说他在问修道院的事。"
  - "他和本地的神殿代表吵了一架。"
  - "他的骑兵往南边搜了一圈，什么都没找到。"
  - "有人在猎人路北边的废墟里看到了光——可能是他在那里。"
```

**T-016 许安然的动静**
```
基础文本：沈清岚放下手中的东西。'许安然那边——有变化。'
触发条件：沈清岚在队中, 距上次此事件>=10天
变体：
  - 修道院的神职人员报告：她最近几天不再重复那句话了。她开始说别的。
  - 或者：周启明的人出现在修道院附近。
  - 或者：她托人传了一张纸条——上面只有一句中文。
```

## 条件系统

每个事件的触发由以下条件组合判定：

```json
{
  "condition": {
    "location_id": [10],           // 必须在灰丘
    "location_type": ["wild"],      // 野外类型
    "time_block": ["下午","傍晚"],   // 这个时段
    "date_range": {"min": 5, "max": 15}, // 每月5-15日
    "min_health": 30,               // 生命>=30
    "max_mental": 60,               // 精神<=60（焦虑时触发的事件）
    "min_affection_with": {"npc_id": 1, "value": 40}, // 好感>=40
    "has_flag": ["已标记A"],        // 必须有这个标记
    "not_flag": ["已完成B"],        // 不能有这个标记
    "has_item": "粮食",             // 必须有这个物品
    "random_chance": 25,            // 25%概率（每时段判定一次）
    "cooldown_days": 3,             // 上次触发后至少3天
    "priority": 3                   // 同时满足多个事件时，取优先级最高者
  }
}
```

### 特殊条件：关系动态

```json
{
  "npc_relationship_dynamic": {
    "npc_a": "沈清岚",
    "npc_b": "蕾娜",
    "condition": "both_in_party",
    "trigger": "他们之间的距离——比上次更远了还是更近了？"
  }
}
```

## 事件生成示例

**同样的模板，不同的参数，完全不同的事件：**

```
模板 T-004：NPC主动搭话

参数集A——
  npc=沈清岚, location=灰丘·傍晚, affection=55
  生成：沈清岚在炉火旁坐了下来。火光把她的侧脸切成了明暗两半。她的手指卷着袖口的线头——那个线头她已经卷了好几天了。'关于许安然，'她说，'我想去见她。'

参数集B——
  npc=格兰, location=沼泽边缘·上午, affection=25
  生成：格兰在前面停了下来。他没有回头，但你听到他的弓弦轻轻响了一下——不是拉满，只是用手指拨了一下。'前面有人来过，'他说，'不是猎人。'
```

## 事件优先级机制

当同一时段多个事件满足条件时：

| 优先级 | 事件类型 | 说明 |
|---|---|---|
| **1 (最高)** | 主线剧情推进 | 周启明行动、许安然变化、红月重合 |
| **2** | 生存危机 | 粮食耗尽、伤口恶化、魔潮逼近 |
| **3** | NPC关键事件 | 感情转折、重大冲突、关键情报 |
| **4** | 资源/交易机会 | 价格波动、意外收获 |
| **5** | 日常随机 | 路边发现、琐事 |
| **6 (最低)** | 纯气氛 | 天气变化、环境描述 |

> 每个时段最多触发**1个主动事件** + **不限数量的被动后果**（如粮食消耗更新）。
