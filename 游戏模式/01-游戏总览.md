# 游戏总览

## 游戏名称

**异世界生存：灰丘领主**

## 核心概念

你是一个被召唤到异世界的无能力高中生。在末日倒计时的压力下，用头脑、契约和微小的资源，在圣光王国的边境建立自己的生存空间。

**游戏类型**：文字冒险 / 资源管理 / 叙事驱动 / 回合制生存

## 核心循环

```
当前状态 → 可选行动 → 你做出选择 → 处理结果 → 触发事件 → 新状态 → 循环
```

每个"回合"游戏主持人（AI）会：
1. 显示当前状态（位置、时间、资源、附近的人）
2. 列出当前可用的行动选项
3. 根据你的选择推进时间、消耗资源、触发事件
4. 返回新的状态

## 时间系统

### 一天7个时段

| # | 时段 | 时间 | 特征 |
|---|---|---|---|
| 1 | **黎明** | 04:00-06:00 | 大多数人还没醒。适合秘密行动。行动后恢复少量体力 |
| 2 | **清晨** | 06:00-08:00 | 一天开始。适合准备、规划、交易 |
| 3 | **上午** | 08:00-12:00 | 主要活动时段。效率最高。可连续做2个行动 |
| 4 | **正午** | 12:00-14:00 | 午餐时间。适合社交。可触发"餐桌对话"事件 |
| 5 | **下午** | 14:00-18:00 | 主要活动时段。效率高 |
| 6 | **傍晚** | 18:00-20:00 | 天色渐暗。室外行动受限。适合室内工作 |
| 7 | **夜晚** | 20:00-04:00 | 必须休息（恢复体力）。除非有特殊事件（逃亡、守夜等） |

### 时段消耗

- 大多数行动消耗**1个时段**
- 长途移动消耗**2-4个时段**（取决于距离）
- 某些大型行动（建设、战斗）消耗**2-3个时段**
- 每天至少消耗**1单位粮食**（每人）

## 资源系统

### 核心资源

| 资源 | 初始值 | 范围 | 说明 |
|---|---|---|---|
| **生命 (HP)** | 100/100 | 0=死亡 | 受伤、疾病、饥饿会降低。休息、药品可恢复 |
| **体力 (SP)** | 100/100 | 0=无法行动 | 每个行动消耗。休息恢复。体力归零则强制休息 |
| **精神 (MP)** | 100/100 | 0=崩溃 | 黑暗事件、压力、恐惧会消耗。0时做决策有概率出错 |
| **银币** | 120 | 0=无法购买 | 基本货币 |
| **铜币** | 0 | 1银=100铜 | 小面额 |
| **粮食** | 7天 | 0=饥饿 | 每天每人消耗1单位。无粮食则HP每天-15 |
| **药品** | 0 | — | 外伤药、退烧药、魔化抑制剂等 |

### 其他追踪项

| 追踪项 | 说明 |
|---|---|
| **位置** | 当前所在地点 |
| **日期** | 光明历847年X月X日 |
| **红月倒计时** | 约700天（每次红月重合约78天，每次写"天空又红了一点"） |
| **声望** | 与各势力的关系值（神殿、苇水村、暮河镇、亚人网络等） |
| **好感度** | 与各NPC的关系值（-100到+100） |

## 行动类型

### 每时段可选行动

| 行动类别 | 具体行动 | 时段消耗 | 说明 |
|---|---|---|---|
| **移动** | 前往XX地点 | 1-4 | 移动到相邻或更远的地点 |
| **工作** | 打工/运输/经营 | 1-2 | 赚取银币 |
| **交易** | 购买/出售/以物易物 | 1 | 在集市或商人处交易 |
| **探索** | 侦察周围/搜寻物资 | 1-2 | 可能发现物品、地点、事件 |
| **社交** | 与XX交谈 | 1 | 推进关系、获取情报 |
| **建设** | 修缮/建造/种植 | 2-3 | 灰丘建设相关 |
| **休息** | 睡觉/养伤 | 1-4 | 恢复体力、生命 |
| **特殊** | 事件限定的独特行动 | 不定 | 根据事件而定 |

## 死亡条件

| 条件 | 结果 |
|---|---|
| **生命归零** | 死亡——游戏结束 |
| **精神归零超过7天** | 精神崩溃——无法控制决策，大概率导致死亡 |
| **连续10天无粮食** | 饿死 |
| **被神殿捕获（特定事件失败）** | 处决或终身监禁——游戏结束 |
| **在魔潮中暴露过久** | 魔化——变成魔化生物——游戏结束 |

## 胜利条件（可选）

| 条件 | 说明 |
|---|---|
| **领地建立** | 灰丘被承认为正式领地 |
| **预言破解** | 发现红月的真相并阻止或延缓 |
| **存活至红月日** | 活到预言中的那一天——无论发生什么 |

> 游戏没有"通关"。你可以选择继续玩下去——经营领地、救援同学、扩展势力——直到你觉得"这个存档该结束了"。
