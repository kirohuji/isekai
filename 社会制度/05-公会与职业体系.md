# 公会与职业体系

## 核心特征

这个世界**没有传统奇幻作品中的"冒险者公会"**。

原因：
- 魔物讨伐被神殿通过勇者体系垄断
- 神殿不容许存在独立于自己的武装
- 任何组织化的冒险者群体都可能威胁神殿对"保护者"角色的独占
- 地方领主可自行雇佣武装，但不允许成立常设的公开冒险者组织

但在民间，各种替代性的组织形式依然存在。

---

## 一、商人公会（Merchant Guild）

王国最强大、组织最完善的民间势力。

| 维度 | 内容 |
|---|---|
| **总部** | 王都商业区 |
| **分会** | 各主要城镇（包括暮河镇设有分会） |
| **核心职能** | 贸易许可发放、价格协定、债务追讨、商队保护协调 |
| **与神殿关系** | 竞争与合作并存。神殿通过第六圣殿管理商业税收，商人公会则试图争取更多自主权。两者在利益上相互需要，在权力上相互提防 |
| **与王室关系** | 商人公会通过贷款和物资供应影响王室决策。国王有时需要商人公会的财力来制衡神殿 |

### 阴暗面

| 行为 | 代表 | 说明 |
|---|---|---|
| **垄断与囤积** | 罗德里克 | 利用末日信息大量囤积粮食和药品，人为推高物价 |
| **欺压亚人商队** | 部分商人 | 拖欠货款、举报亚人"私藏逃奴"以逃避债务 |
| **与地方权力勾结** | 罗德里克贿赂税务官和守卫队长 | 证明商人与地方官员的利益共生关系 |

### 与男主阵营的关系

| 当前 | 潜在 |
|---|---|
| 无直接接触 | 如果灰丘发展出贸易能力，可能需要与商人公会打交道。罗德里克是商人公会成员——这可能是冲突点也可能是入口 |

## 二、工匠公会（Craftsman Guild）

| 维度 | 内容 |
|---|---|
| **组成** | 铁匠行会、木匠行会、石匠行会、皮匠行会、药剂师行会等 |
| **组织方式** | 师徒制，技术传承封闭。各城镇有各自的行业师傅 |
| **社会地位** | 中级——低于商人公会，但高于普通农民 |
| **末日价值** | 🔴 武器锻造、建筑修缮、药品制作——都是末日生存的核心需求 |

### 与男主阵营的关系

| 当前 | 潜在 |
|---|---|
| 间接关联——王都某药铺的药剂师学徒曾帮玛莎取得伪造的传染病证明 | 灰丘建设需要木匠和石匠。未来可能需要接触暮河镇的工匠 |

---

## 三、猎人协会（Hunter Association）——半合法组织

| 维度 | 内容 |
|---|---|
| **法律地位** | **灰色地带。** 官方不承认但也不积极取缔——因为边境地区需要猎人控制野兽和部分低级魔物 |
| **主要活动区域** | 东南沼泽、北方边境等王国控制力薄弱的地区 |
| **成员构成** | 大量亚人（特别是兽耳族猎人）、边境人类猎户、退役士兵 |
| **职能** | 狩猎、毛皮和药草贸易、低级魔物清除、向导服务、失踪者搜寻 |
| **与神殿关系** | 神殿默许存在，前提是不挑战勇者体系。如猎人协会开始组织化讨伐高级魔物，神殿会立即镇压 |
| **与勇者关系** | 个别勇者有时雇佣猎人做向导或追踪。但整体上是上下关系——猎人在勇者面前没有平等的地位 |

> **格兰**不是任何组织的正式成员，但他与猎人协会的亚人网络保持联系。他更接近"独立猎人"——不依附组织，但利用组织的信息和人脉。

## 四、佣兵团（Mercenary Companies）

| 维度 | 内容 |
|---|---|
| **性质** | 私人武装，受雇于贵族进行小规模战争、边境冲突、矿区镇压或商队保护 |
| **规模** | 数十人到上百人不等。最大者或有数百人 |
| **法律地位** | 合法——只要能取得地方领主的书面雇佣许可 |
| **可靠性** | 低。今天为你战斗的佣兵，明天可能接受你敌人的雇佣。忠诚取决于报酬是否按时发放 |
| **末日中的风险** | 在物资短缺时，佣兵团很容易变成武装匪帮——抢劫村庄比等待佣金更快 |

## 五、地下组织

### 5.1 走私网络

- 由亚人商队（特别是鼠族）和少数边境人类共同运作
- 绕过神殿税收和王国防线的秘密贸易路线
- 主营：药品、武器、盐、亚人特产（如银叶草）、禁止贸易品
- 男主的亚人网络连接部分依赖走私渠道

### 5.2 奴隶贩运网络

- 灰绳市场是合法门面
- 背后涉及：人口绑架、逃奴追捕、异界女性（召唤者）的地下交易
- 神殿对此选择性执法——神殿自身就是女性召唤者的最大消费者之一

### 5.3 反神殿秘密团体（待展开）

- **推测存在**：部分对神殿不满的贵族、商人、低级神职人员可能组成秘密网络
- **风险极高**：此类团体往往是神殿渗透的优先目标。接触任何自称"反神殿"的人都可能是陷阱
- **潜在价值**：如果存在真实的团体，可能掌握神殿的内部情报

## 六、职业与社会地位总览

| 阶层 | 典型职业 | 社会地位 |
|---|---|---|
| **统治层** | 国王、大贵族、边境伯爵 | 最高 |
| **宗教层** | 大神官、枢机祭司、地方祭司 | 最高 |
| **军事精英层** | 核心勇者、高级将领、神殿骑士 | 高 |
| **商业层** | 大商人、药材商、船主 | 中高 |
| **技术层** | 高级工匠、药剂师、抄写员、账房 | 中 |
| **平民层** | 农民、低级工匠、猎人、小商贩、普通士兵 | 低 |
| **贱民层** | 奴隶、逃奴、无身份流民、亚人（法律地位低下） | 最低 |

## 七、与男主阵营的交互矩阵

| 组织 | 当前关系 | 潜在价值 | 风险 |
|---|---|---|---|
| **商人公会** | 无接触 | 贸易渠道、物资采购、情报 | 罗德里克是成员；可能向神殿举报 |
| **工匠公会** | 间接（药剂师学徒） | 灰丘建设所需技术 | 低——工匠不问政治 |
| **猎人协会** | 有限接触（通过格兰） | 武力补充、向导、边境情报 | 亚人不信任人类 |
| **佣兵团** | 无接触 | 未来可能的武力来源 | 不可靠、成本高、易反噬 |
| **走私网络** | 有限（通过格兰/鼠族） | 绕过封锁获取物资的唯一途径 | 与罪犯绑定 |
| **反神殿团体** | 无接触 | 情报、潜在盟友 | 可能被神殿渗透——致命风险 |

## 八、设定原则

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| 把任何组织写成无条件帮助男主 | 每个组织都有自己的利益和底线 |
| 把"公会"写成万能任务板 | 这个世界的组织是灰色的、政治的、需要经营关系的 |
| 靠加入某个组织解决所有问题 | 男主的核心路径是建立自己的组织（灰丘），而非依附于现有体系 |
| 组织间关系简单化 | 组织间的关系是流动的——今天的合作伙伴可能因为一次利益冲突变成敌人 |
