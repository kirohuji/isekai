# 无能力者的价值

## 核心悖论

> 一个在任何力量鉴定中都是零分的人，却是最被神殿「持续观察」的对象。

## 男主的非力量能力体系

| 能力维度 | 具体表现 | 用途场景 |
|---|---|---|
| **语言学习** | 花20银币掌握王国通用语 | 经营、谈判、法律沟通 |
| **记账与成本计算** | 旅馆经营、物资规划 | 经济管理 |
| **制度漏洞识别** | 利用神殿搜查的规则空隙 | 藏匿沈清岚 |
| **商业经营** | 物流运输、旅馆分期接手 | 独立生存 |
| **物流组织** | 救济运输、以粮雇佣饥民 | 危机中的资源调配 |
| **谈判** | 与格兰建立利益同盟 | 无武力下的合作 |
| **危机拆解** | 四层撤离计划 | 复杂行动的分步执行 |
| **责任承担** | 不放弃被买下的沈清岚 | 建立长期信任 |
| **风险判断** | 拒绝赴旧钟楼之约 | 识别陷阱 |

## 「无能力者」的类型细分

故事中的无能力者并非只有男主一人。其他无能力者的处境可以展示这个制度的残酷性：

| 类型 | 代表 | 处境 |
|---|---|---|
| **组织型** | 男主 | 被神殿观察，靠自己建立生存空间 |
| **被放弃型** | 部分无能力同学 | 获得安置金后被神殿放弃，自生自灭 |
| **被奴役型** | 无能力的女性同学 | 以「保护」为名沦为性奴隶 |
| **依附型** | — | 依附勇者或神殿，换取生存保障 |

## 设定原则

> ❌ 男主不能突然获得万能战斗能力。
> ✅ 男主的优势始终是组织、商业、信息、判断和责任感。

男主代表了另一种力量路径：
- 在一个只看力量等级的世界里，他用**信任、契约和信息**建立了真实的生存基础
- 他的「力量」不容易被鉴定，也不容易被复制
- 这恰恰是神殿「持续观察」的原因——神殿不确定如何评估和利用这种人
