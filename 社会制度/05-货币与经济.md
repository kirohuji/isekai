## 末日经济趋势

边境正在进入**灾荒经济**，以下物资持续升值：

```
粮食 → 退烧药 → 伤药 → 魔物解毒剂 → 盐 → 干燥木材 → 运输能力
```

### 原因

- **粮食**：自然灾害+战时征收（每村上交三成粮食）
- **药品**：魔物增多导致受伤和中毒增加，罗德里克囤积
- **盐**：保存食物的必需品
- **木材/运输**：建设和物流的基础
- **金银**：在末日经济中贬值，实物资源升值
