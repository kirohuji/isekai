# 社会结构

王国是典型的**封建神权国家**。

## 权力层级

| 层级 | 组成 | 说明 |
|---|---|---|
| **第一层** | 国王与王室 | 名义上的国家最高权力，实际受神殿制约 |
| **第二层** | 神殿与女神信仰体系 | 掌握召唤、祝福、预言、身份登记、奴隶契约、部分司法权和大量土地 |
| **第三层** | 大贵族、边境伯爵、地方领主 | 地方实权者，与神殿既有合作也有竞争 |
| **第四层** | 被承认的勇者 | 法律上受王室与神殿共同保护的特殊阶层 |
| **第五层** | 商人、公会、自由民 | 社会中间层 |
| **第六层** | 农民、矿工、雇工 | 底层劳动者 |
| **第七层** | 奴隶、逃奴、无身份流民、部分亚人 | 最底层 |

## 关键制度特征

- **神殿不限于宗教**：同时掌握司法、土地、身份登记、奴隶契约和召唤仪式
- **勇者是制度工具**：不仅用于对抗魔王，也是王国和神殿统治普通召唤者的工具
- **奴隶制合法**：奴隶契约受法律保护，窝藏逃奴可致死刑
- **亚人地位低下**：在人类王国中权利受限，但在边境地区和沼泽地带拥有自己的网络
