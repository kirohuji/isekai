# 末日经济学

## 核心原则

> 金银不再是最重要的通货。末日时代最值钱的资源是：**土地 > 水 > 粮食 > 药品 > 交通 > 情报 > 可靠的人**。

## 各类物资详解

### 土地（最底层/最基础）

- 灰丘：一块荒废七年的废弃高地
- 苇水村的土地：提供耕种可能
- 末日中土地的价值在于：可以自给自足，不依赖外部供应链

### 水

- 灰丘已有初步雨水收集
- 沼泽水源可能受到魔化污染
- 河水在魔潮后可能不安全

### 粮食（当前最紧迫）

| 状态 | 说明 |
|---|---|
| 男主储备 | 零（完全依赖苇水村） |
| 苇水村储备 | 仅余十余天 |
| 王都征收令 | 可能每村上交三成 |
| 罗德里克囤积 | 进一步推高粮价 |

### 药品

| 药品类型 | 状态 | 紧迫度 |
|---|---|---|
| 退烧药 | 无储备 | 🔴 |
| 伤药 | 无储备（仅男主处理伤口的基础物资） | 🔴 |
| 魔物解毒剂 | **被罗德里克囤积** | 🔴 |
| 米娅腿伤用药 | 少量 | 🟡 |

### 交通/运输

- 男主失去骡车后无运输工具
- 格兰知道走私路线和秘密通道
- 运输能力 = 获取远方资源的能力

### 情报（顶层）

- 沈清岚的名单：50名同学的去向
- 许安然的预言：红月倒计时
- 格兰的亚人网络：边境动向
- 罗德里克账册：地方腐败证据

### 可靠的人

- 当前4人：男主、沈清岚、格兰、米娅
- 苇水村：善意但有限
- 旧旅馆员工：失联
- 最稀缺的资源——无法用钱买

## 通货膨胀逻辑

| 资源 | 价值变化 | 原因 |
|---|---|---|
| 粮食 | 📈 快速上涨 | 自然灾害+战时征收+商人囤积 |
| 药品 | 📈 快速上涨 | 魔物增加+囤积 |
| 盐 | 📈 上涨 | 保存食物的必需品 |
| 运输 | 📈 上涨 | 道路危险+封锁 |
| 金银 | 📉 贬值 | 实物资源更有价值 |
| 契约/身份 | 📈 上涨 | 合法保护的重要性上升 |
