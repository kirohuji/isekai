# 神殿

## 核心职能

| 职能 | 说明 |
|---|---|
| 召唤异世界人 | 周期性进行，非第一次发生 |
| 勇者祝福 | 赋予选中的召唤者以神力 |
| 预言记录与解读 | 许安然等被分配至此 |
| 身份登记与管理 | 控制召唤者去向 |
| 奴隶契约管理 | 合法化人口买卖 |
| 部分司法权 | 可审判逃奴和异端 |
| 大量土地持有 | 经济基础 |

---

## 组织架构：七大神殿

神殿以「**七大圣殿**」为构架，每座圣殿负责不同领域：

| 圣殿 | 所在地 | 核心职能 | 与故事当前关联 |
|---|---|---|---|
| **第一圣殿** | 王都中心 | 最高决策、大神官驻地、总领一切神殿事务 | 整个神殿体系的核心 |
| **第二圣殿** | 王都 | 军事事务、勇者祝福仪式、军队随军祭司 | 周启明等核心勇者与此殿关联 |
| **第三圣殿** | 王都 | 档案管理、历史记录、召唤记录、财务账册 | **沈清岚曾在此抄写**——发现周期性召唤的秘密 |
| **第四圣殿** | 未知（推测北部） | 异端审判、逃犯追捕、异世界检测铃的管理与研发 | 负责追捕沈清岚的机构 |
| **第五圣殿** | 分布于各地 | 农业管理、土地登记、税收、粮食储备 | 战时征收令的执行者 |
| **第六圣殿** | 多地（含暮河镇） | 商业管理、奴隶契约认证、贸易许可发放 | 与商人公会、罗德里克等关系密切 |
| **第七圣殿** | 未知（推测边境） | 预言解读、神秘学研究、古代文献保管 | **许安然被分配至此** |

## 主要目标（推测）

| 目标 | 证据 |
|---|---|
| 控制召唤者 | 能力鉴定后将人分到不同岗位；女性被集中为性奴隶 |
| 隐瞒召唤真相 | 周期性召唤的秘密被沈清岚发现 |
| 保持神殿权威 | 以「十年备战」维持征税征粮的合法性 |
| 回收沈清岚及名单 | 周启明追踪、旅馆使用异世界检测铃搜查 |
| 观察男主 | 神殿对男主的记录：「暂不接触，持续观察」 |

## 神殿可能隐瞒的真相

1. 维持勇者不可战胜的形象——防止民众恐慌
2. 防止民众提前逃亡——避免王国秩序崩溃
3. 以「十年备战」为名继续征税征粮——维持神殿财力
4. 隐瞒神殿无法阻止魔王苏醒——承认无能会动摇权威
5. ⚠️ **召唤仪式可能与魔王苏醒存在因果关系**——仅可能性，尚无证据

> 第5点是整个故事最深的悬念之一：如果神殿的召唤仪式本身就在唤醒魔王，那么整个「召唤勇者对抗魔王」的叙事就是彻底的谎言。

## 神殿对男主的特殊态度

男主的鉴定记录并非「无价值」，而是：

> **「暂不接触，持续观察」**

这意味着神殿认为男主存在**未知价值或风险**。可能原因：
- 无能力却能在异世界生存和组织，本身就是异常
- 神殿可能预见到组织型人才的长期价值或威胁
- 与召唤秘密有关

## 异世界检测铃

- 神殿用于追踪召唤者的魔法道具
- 在王都搜查中曾用于旧鹿角旅馆
- 男主用自己登记过的召唤者身份和异世界物品成功干扰检测
