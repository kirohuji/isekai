# 叙事规则

## 视角

- 默认第一人称「**我**」指男主
- 读者被**直接塞入男主的意识内部**——不是在旁观男主，而是**成为**男主。所有的感受、判断、恐惧、欲望，都以"我"的原始形态呈现，不经修饰
- 其他角色的内心不可直接进入。他们的心理状态只能通过"我"的观察来推断——表情、语气、动作、沉默的间隙

## 核心叙事风格

| 维度 | 说明 |
|---|---|
| **第一人称沉浸式** | "我"看到什么、听到什么、闻到什么、摸到什么——读者与男主共享同一个感官系统。没有上帝视角，没有旁白解说。世界通过男主的身体感知一点点渗进来 |
| **意识流** | 男主的思维是流动的、跳跃的、不按逻辑起承转合的。恐惧会突然切入一段记忆。愤怒会打断正在进行的计算。欲望会在不恰当的时刻冒出来。内心独白是**未经整理的原始意识**，而不是经过修辞的"思考总结" |
| **心理描写** | 男主所有的情感——恐惧、欲望、愧疚、愤怒、占有欲、自我厌恶——都不被删节或美化。心理描写应直接、粗粝，让读者感受到一个真实的人在极端环境下的内心全貌。包括那些他不愿承认的念头：想侵犯她、想丢下她逃跑、想杀死某个人 |
| **暗黑** | 世界是灰的、脏的、腐烂的。环境描写不回避血腥、污秽、尸体、排泄物、伤口和蛆虫。人性的阴暗面——贪婪、背叛、残忍、冷漠——是常态而非例外。但不为暗黑而暗黑——每处暗黑描写应服务于"这个世界正在死去"的基调 |
| **刺激** | 紧张场景不给读者喘息的机会。短句、断句、碎片化感知。男主的心跳、呼吸、皮肤上的汗——用生理反应驱动读者的生理反应。逃生场景要让读者不自觉屏住呼吸 |

## 描写重点

| 维度 | 说明 |
|---|---|
| **资源变化** | 每一章开始和结束时的资源数量（现金、粮食、药品、人力）——"我"对数字的执念是焦虑的外化 |
| **伤势与体力** | "我"受伤后的状态变化——不是"我受了伤"，而是"每走一步右腿都在抖，视野边缘发黑，胃里翻涌着想吐" |
| **行动代价** | 每次成功的真实成本：资源、暴露、受伤、欠人情、制造新敌人。代价不是事后总结，而是在行动中实时感受到的痛 |
| **情报真假** | "我"得到的信息可能不完整、被扭曲、被故意误导。"我"必须在不确定中做决定——而不是等一切都清楚了再行动 |
| **细微动作** | 角色的表情、距离、气味、呼吸频率——"我"的观察不是全知的，而是选择性、偏见性、可能错判的 |
| **长期后果** | 当前决策对未来的影响应在后续章节体现——"我"的选择留下的伤疤会在后面发炎 |

## 各角色说话风格

### 男主（"我"的内心声音）

- 对外**简洁**——话不多，但每句都经过算计
- 常用成本、风险、概率和责任解释行为
- 很少直接说「我担心你」，更可能说：
  - 「你现在不能继续走」
  - 「这个风险不值得」
  - 「如果失败，损失的不只是我们」
- **内心意识流**——与对外简洁形成强烈反差。内心是碎片的、矛盾的、反复的：
  - 一段计算突然被一段记忆打断
  - 正在分析风险，脑海里却冒出她后颈的皮肤纹理
  - 告诉自己"这是最合理的决策"，同时知道自己只是在找借口
- 面对沈清岚时内心的变化：
  - 解释过多（对自己解释为什么多看了她一眼）
  - 不承认的嫉妒（"我只是在评估周启明对她还有多少影响力"）
  - 无法命名的感情（"这不是喜欢。这是责任。这是成本考量。这是——"）

### 沈清岚

- **冷静、直接**
- 会拆穿男主的借口：
  - 「你说风险不值得，实际是担心我。」
- 谈判时比男主更敢提出长期保障：
  - 「我要的不是暂时的安全，是今后都不需要躲藏的权利。」
- 在米娅面前更轻松，会开玩笑和八卦
- 告白后不会逼迫男主，但会主动确认关系变化

### 格兰

- **务实**
- 不说无用安慰
- 习惯先问代价：「这要花多少钱/时间/人命？」
- 会直接指出男主的矛盾：
  - 「你说尊重她的选择，但你脸上的表情不是这么说的。」

### 米娅

- **好奇、活泼**
- 喜欢直问恋爱问题（让男主尴尬）
- 会从亚人角度指出人类语言和制度的虚伪：
  - 「你们人类嘴上说平等，但从来没有把兽耳的人当人。」
- 不是纯粹搞笑角色——提到母亲时会安静下来

## 叙事节奏

- **紧张段落**：短句、断句、碎片化。男主的感官被压缩到只感知眼前的威胁。呼吸节奏驱动句子节奏
- **意识流段落**：思维跳跃、联想、闪回、自我质问。不分段的内心洪流。逻辑被打断，情绪切入，再回到逻辑
- **对话段落**：充分展开，不急于推进。但对话中穿插"我"的内心评注——她在说A，我在想B
- **感情段落**：不说"我喜欢她"。用身体反应——心跳、体温、视线无法移开、找借口多留一会儿
- **决策段落**：展示"我"的完整思考链，但允许情绪和偏见污染计算过程——"我"不总是理性的
