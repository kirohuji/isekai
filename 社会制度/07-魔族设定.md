# 魔族设定

## 魔族的本质

魔族不是"被创造出来的邪恶物种"。他们是这个世界的**原住民之一**——与精灵和亚人一样古老。

人类和神殿将魔族定义为"魔王的爪牙"，但真相可能更复杂：**魔族有自己的社会、文化和生存逻辑。** 他们侵略人类王国不是因为"天生邪恶"——是因为他们的生存空间正在被某种力量压缩。而这种力量……可能与神殿的召唤仪式有关。

> ⚠️ 保持"可能的真相"的模糊性——不要急于确定魔族是"正义"还是"邪恶"。他们是**另一个立场**。

## 魔族的分类

### 总览

| 层级 | 分类 | 智力 | 寿命 | 说明 |
|---|---|---|---|---|
| **顶层** | 魔王 | ？ | 近乎永生 | 魔族的最高存在 |
| **高层** | 魔将 | 极高 | 500年以上 | 魔王麾下的指挥官 |
| **中层** | 上位魔族（含魅魔） | 高 | 300-500年 | 具有独立意志和特殊能力 |
| **下层** | 下位魔族 | 低-中 | 50-100年 | 战斗消耗品 |
| **底层** | 魔化生物 | 无-低 | 数月到数年 | 不是真正的魔族——是被魔化的其他生物 |

## 一、魔王（Demon King）

| 维度 | 内容 |
|---|---|
| **本质** | **最大的未解之谜。** 魔王是一个实体存在？一种力量现象？一个被污染的远古存在？——当前不设定明确答案 |
| **已知信息** | 位于北方荒野深处。力量通过"魔潮"向外渗透。能被"感知"但不能被"目击"——所有声称见过魔王的人都没有活着回来 |
| **与召唤仪式的关系** | ⚠️ 最大悬念：召唤仪式可能**在唤醒魔王**或**在维持对魔王的封印**——两种可能性完全相反，神殿坚持后者，但沈清岚发现的记录暗示前者 |
| **叙事功能** | 魔王在本作中是"地平线上的黑暗"——永远在那里，永远在逼近，但不在早期剧情中正面对决 |

---

## 二、魔将（Demon Generals）

魔王麾下的最高指挥官。数量未明确——当前推测为**七位**，与人间的七名核心勇者形成呼应（这是神殿叙事的一部分："七勇者对七魔将"——但这个叙事本身可能是神殿编造的）。

| 维度 | 内容 |
|---|---|
| **力量水平** | 远在核心勇者之上。每一位魔将都相当于一支军队 |
| **个性** | 魔将不是统一阵营。他们有自己的野心、彼此之间有竞争和背叛。部分魔将对魔王的忠诚可能是被迫的 |
| **已知信息** | 亚人老祭司的记录中提到过"北方的七座黑塔"——可能与魔将有关。具体情况未知 |

---

## 三、上位魔族（Greater Demons）

具有独立意志、高等智慧和独特能力的中坚力量。

### 3.1 魅魔（Succubus）——重点设计

| 维度 | 内容 |
|---|---|
| **性别** | 仅有女性形态。对应的男性形态称为**淫魔**（Incubus），但数量远少于魅魔 |
| **外观** | 可切换两种形态：**伪装形态**——外表与人类女性无异（可能极其美丽，也可能极为普通——取决于任务需要）。**真身形态**——皮肤转为暗紫色或深红色，瞳孔变为竖瞳，背后展开薄膜翼，头生小角。真身形态下身体素质远超人类 |
| **核心能力** | **精力吸取。** 通过与男性（淫魔则通过女性）进行性行为，吸取对方的生命力、精气和/或神力。被吸取者会感到极度疲倦、虚弱——严重时可致死。吸取的能量转化为魅魔自身的力量——这使得魅魔是魔族中**唯一可以通过"进食"不断变强的类型** |
| **特殊应用** | 在吸取对象为神力持有者（勇者、能力者）时，魅魔可以**暂时获得对象的部分神力属性**。这是她们作为间谍和渗透者的核心价值——一个成功渗透进神殿的魅魔可以伪装成一名"女祭司"，甚至展现出女神祝福的力量 |
| **社会地位** | 在魔族内部的地位**复杂且矛盾**。一方面：她们是魔王最有价值的间谍和渗透工具。另一方面：其他上位魔族鄙视她们——因为她们的力量是"偷来的"而非"天生的"。魅魔自己也清楚这一点——这导致部分魅魔对魔族本身也抱有疏离感 |
| **弱点** | 精力吸取的欲望是**生理需要**，不是可以无限压抑的——长期不吸取会导致自身衰弱甚至死亡。这使得魅魔无法"选择不害人"——她们必须害人才能活。这一生存困境使魅魔成为魔族中**最复杂的角色类型**——不是纯恶，而是生来就以他人为食的悲剧性存在 |
| **叙事潜力** | 一个试图摆脱吸取本能的魅魔、一个爱上了"猎物"的魅魔、一个被神殿捕获并被迫提供情报的双面间谍——魅魔角色拥有极丰富的叙事可能性 |

#### 魅魔的等级

| 等级 | 特征 | 吸取能力 | 地位 |
|---|---|---|---|
| **魅魔女王** | 极为罕见。据说只有一位——魔王的直属。外表与人类完全无异，即使是神殿检测铃也无法识别 | 可在不进行性行为的情况下远距离吸取 | 接近魔将级别 |
| **高阶魅魔** | 拥有完整双形态切换能力。可长期维持伪装形态 | 单次吸取可致死。可从神力者身上获取临时神力 | 魔王直属间谍 |
| **普通魅魔** | 伪装形态有微小破绽（体温稍低、影子偶尔异样）。需定期吸取 | 吸取致虚弱但不一定致死 | 中层渗透者 |
| **低阶魅魔** | 无伪装能力，真身形态活动。更像"诱饵" | 吸取力较弱，无法处理神力者 | 底层 |

### 3.2 其他上位魔族类型

| 类型 | 特征 | 能力 |
|---|---|---|
| **影魔**（Shadow Demon） | 半实体半阴影的存在。可融入黑暗中移动。智商高，常用作斥候和信使 | 暗影穿行、信息传递、暗杀 |
| **炎魔**（Flame Demon） | 身体由暗色火焰和熔岩状物质构成。体型巨大，极度暴力 | 火焰操控、范围破坏 |
| **冰魔**（Frost Demon） | 苍白皮肤，散发寒气。主要活动在北部寒冷地区 | 冰冻、低温光环 |
| **契约魔**（Pact Demon） | 擅长与人签订契约——以力量换取忠诚或灵魂。是魔族与人类之间最危险的"中间人" | 契约魔法、心理操控、交易 |
| **脑魔**（Mind Demon） | 小型，无肢体，外形接近暴露的大脑。精神操控专精 | 心灵感应、记忆读取、精神控制 |

## 四、下位魔族（Lesser Demons）

低智力或中等智力的魔族战士。是魔潮中的主力。

| 类型 | 特征 |
|---|---|
| **魔兵**（Demon Soldier） | 人形魔物，持武器。有基础战术能力。皮肤暗灰至黑色 |
| **魔犬**（Demon Hound） | 四足猎食者。群体行动。嗅觉追踪极强 |
| **翼魔**（Winged Demon） | 小型飞行魔物。群攻型。单独时很弱 |
| **巨魔**（Troll-type） | 体型巨大（3-5米），力量恐怖，智商低。需要魔将或上位魔族的指挥才能有效作战 |

## 五、魔化生物（Corrupted Beings）

> ⚠️ **魔化生物不是魔族。** 它们是被魔王的力量污染后异变的普通生物——包括动物、植物，以及人类。

| 类型 | 说明 |
|---|---|
| **魔化动物** | 腐齿鼠、腐兽、沼鳄等。特征是黑鳞和魔化晶体。行为异常——攻击性激增，不再遵循自然习性 |
| **魔化人类** | 被魔潮严重污染的人类。过程缓慢但不可逆（目前）。保留部分智力但失去自制力。是"魔化后的人类还能不能算人"这一伦理问题的载体 |
| **魔化植物** | 异常生长、散发有毒孢子、根系能移动。将一片区域变成"魔化森林" |

魔化的详细阶段见 `社会制度/04-能力体系详解.md`。

## 六、魔族的社会结构

### 是否有一个"魔族王国"？

不完全。魔族的世界更像是一个**以魔王为中心的松散等级秩序**，而不是一个中央集权国家。

- 魔王是核心。魔将各自掌控一片领地（北方荒野中的不同区域）
- 魔将之间竞争激烈——魔王的注意力是有限的资源
- 上位魔族（如魅魔）在某些魔将麾下服务，也可以在不同魔将之间流动
- 存在**不受魔王直接控制的独立魔族群体**——在北方荒野的偏远角落，有小型魔族聚落自行生存，不参与对人类的战争

### 魔族为什么要南侵？

神殿的叙事：魔族天性邪恶，以毁灭秩序为乐。

可能的真相（多重可并行）：
| 可能性 | 说明 |
|---|---|
| **生存空间被压缩** | 北方正在发生某种恶化（可能与魔潮的源头有关），魔族被迫南迁 |
| **魔王的意志** | 魔王确实在主动发动战争——但原因未知 |
| **召唤仪式的反噬** | 每次神殿召唤异世界人，都会在北方产生一次"能量震荡"——这种震荡对魔族的影响是毁灭性的。魔族南侵可能是对召唤的直接反应 |
| **源流的失衡** | 亚人的源流信仰认为——女神（光明）和魔王（黑暗）是源流的两面。神殿过度强化"光明"一方（召唤、祝福），导致源流失衡——魔王一侧的力量被迫向外溢出 |

## 七、魔族与人类的接触形式

| 形式 | 说明 | 已知案例 |
|---|---|---|
| **魔潮异变** | 被动接触——魔王力量渗透导致生态异变 | 苇水村洪水、河水倒流 |
| **魔化生物袭击** | 被污染的生物攻击人类 | 腐兽袭击婴儿 |
| **前线战斗** | 北方叹息之墙附近，人类守军与魔族的实际交战 | 陈浩等镇压活动间接与此相关 |
| **间谍渗透** | 魅魔或契约魔潜入人类社会，以伪装身份活动 | **当前无已知案例——但这是重要的叙事潜力** |
| **秘密交易** | 部分人类（可能是商人、贵族甚至神殿高层）可能与魔族有秘密接触——以情报换利益 | **纯推测，无证据** |

## 八、设定原则

| ❌ 禁止 | ✅ 正确方向 |
|---|---|
| 把魔族写成铁板一块的邪恶阵营 | 魔族内部有派系、有独立的个体选择、有复杂的动机 |
| 把魅魔写成纯粹色情角色 | 魅魔是精力吸取的悲剧性存在——她们的生存方式决定了她们无法"不害人"，但这不意味着她们享受害人 |
| 魔族=丑陋怪物 | 上位魔族（特别是魅魔）拥有与人类无异的外表。魔族的恐怖不在于"长得丑"，而在于"看不出来" |
| 魔王早期出现 | 魔王是"地平线上的黑暗"——永远在逼近，但不在早期剧情中成为直接对手 |
| 把所有故事问题归因于魔族 | 人类的敌人不仅是魔族——还有彼此。神殿、腐败贵族、各国之间的内斗——这些在末日中不会消失 |
